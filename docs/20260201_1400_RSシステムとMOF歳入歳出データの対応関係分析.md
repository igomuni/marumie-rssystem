# RSシステムとMOF歳入歳出データの対応関係分析

## 調査目的

府省庁単位でRSシステムの事業予算を合計した金額と、MOF（財務省）データの歳入・歳出の金額の対応関係を調査する。

### RSシステムデータ
- `data/year_2024/2-1_RS_2024_予算・執行_サマリ.csv`: 予算・執行のサマリ情報
- `data/year_2024/2-2_RS_2024_予算・執行_予算種別・歳出予算項目.csv`: 予算種別・会計区分別の詳細

- `data/download/mof_2024/DL2024*.csv`: 財務省公開の予算データ

## 差異の原因分析

### RSシステムとMOF歳出の差異（約435兆円）の主な要因

1. **RSシステムの対象範囲の限定**
   - RSシステムは「行政事業レビュー」対象の事業のみを計上
   - 以下の予算はRSシステムに含まれない（または部分的にしか含まれない）:
     - 国債費（約28兆円）
     - 地方交付税交付金等（約16兆円）
     - 人件費の大部分
     - 年金・医療等の義務的経費の一部

2. **特別会計の取り扱い**
   - MOFデータには特別会計間の繰入・繰出が含まれる
   - 同一の資金が複数回カウントされる（グロス計上）
   - RSシステムは実質的な事業予算（ネット計上に近い）

3. **府省庁名のマッピングの違い**
   - RSシステム: 外局（林野庁、水産庁など）を独立して計上
   - MOFデータ: 外局は親省庁に統合、または複数省庁共管として計上
   - 例: RSの「林野庁」「水産庁」→ MOFの「農林水産省」

4. **会計処理の違い**
   - MOFの「財務省」285兆円には国債の償還・利払いが含まれる
   - MOFの「総務省」36兆円には地方交付税が含まれる

## 重要な知見

### 1. RSシステムの位置づけ
- RSシステムは「事業単位」で予算を管理する仕組み
- 政策評価・行政事業レビューの対象となる「事業」に限定
- 国の予算全体（一般会計＋特別会計の純計約250兆円）の約48%をカバー

### 2. MOFデータの特性
- **歳出**: 所管別・組織別・目別の詳細な予算配分
- **歳入**: 主管別・款別の収入構成
- 特別会計を含むため、会計間の資金移動が重複計上される

### 3. 対応関係の限界
- 両データは目的・粒度が異なるため、完全な1対1対応は困難
- 府省庁名の表記・分類が異なる
- RSは「事業」単位、MOFは「予算科目」単位

## 可視化への示唆

### RSシステムの可視化における注意点

1. **財源情報の扱い**
   - RSシステムは財源情報を詳細に持っていない
   - MOFの歳入データとの直接的なリンクは困難
   - 「一般会計」「特別会計」の区分程度の財源表示が現実的

2. **金額の解釈**
   - RSシステムの146兆円は「事業予算」であり、国の予算全体ではない
   - ユーザーへの注釈として「行政事業レビュー対象事業の予算」であることを明示すべき

3. **府省庁別集計の注意**
   - RSシステムの「厚生労働省」91兆円は、年金特会等を含む数値
   - これはMOFの「厚生労働省」72兆円より大きい
   - 社会保障関連の特別会計がRSでは厚労省に帰属している

## 結論

RSシステムとMOFデータは、それぞれ異なる目的で整備されたデータであり、直接的な対応関係を取るのは困難である。ただし、以下の形での活用は可能:

1. **補完的な利用**: MOFの歳入構成（租税・公債金など）を財源の「文脈情報」として提示
2. **会計区分の活用**: RSシステムの「一般会計」「特別会計」区分を活用した財源別表示
3. **注釈の付与**: 可視化時に「行政事業レビュー対象事業」であることを明記
4. **年度間推移の活用**: 府省庁別・会計区分別の年度推移を可視化することで、政策変化を把握可能
