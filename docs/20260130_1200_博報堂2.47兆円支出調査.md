## 4. 博報堂の役割分析

### 4.1 事務局（Secretariat）としての役割

博報堂は本事業の「事務局」として以下の機能を担っている：

1. **補助金の審査・交付事務** - 中小電力・ガス小売事業者からの申請受付、審査、補助金交付
2. **値引き原資の配分** - 小売事業者への補助金（値引き原資）の配分・管理
3. **広報・周知活動** - 制度の周知、消費者への広報
4. **システム運用** - 申請・管理システムの構築・運用
5. **コールセンター運営** - 問い合わせ対応

### 4.2 支出の二重構造

本事業では、補助金の配分に2つのルートが存在する：

```
                    ┌─────────────────────────────────────────────┐
                    │                                             │
                    ▼                                             │
経済産業省 ───→ 大手電力・ガス会社（17社）──────→ 消費者への値引き
    │            （直接補助：2.02兆円）                  │
    │                                                    │
    └─────→ 株式会社博報堂（事務局）───────→ 中小小売事業者 ──→ 消費者への値引き
             （2.37兆円）                  （パススルー）
                 │
                 └─→ 事務局運営経費（博報堂の取り分）
```

## 8. 考察と論点

### 8.1 透明性の問題

- 2.37兆円という巨額の支出において、事務局経費とパススルー補助金の内訳が開示されていない
- 行政事業レビューシステムのデータ構造上、この区別ができない設計になっている

### 8.2 事務局選定の適切性

- 広告代理店である博報堂が、電力・ガス補助金事業の事務局として選定された理由は、データからは読み取れない
- 随意契約(その他)や補助金等交付の形式で多額の契約が行われている

### 8.3 他の類似事業との比較

- マイナポイント事業では一般社団法人キャッシュレス推進協議会が事務局（4,967億円）
- IT導入補助金ではサービスデザイン推進協議会が事務局（710億円）
- いずれも事務局経費とパススルー補助金の区別は明確ではない

## 9. まとめ

### 主要な発見

1. **博報堂への約2.9兆円のうち、パススルー補助金が2兆9,062億円（98.4%）**
   - 費目・使途データにより、内訳が明確に判明
   - 中小電力・ガス小売事業者への値引き原資として配分

2. **博報堂グループの実際の事務局経費は約297億円（1.0%）**

   - 事務費: 27億円

   - 一般管理費: 2.8億円

3. **博報堂の純粋な人件費はわずか7.9億円**

   - 博報堂は「元請け・統括」機能を担い、実務は再委託先が実施

4. **電力・ガス会社への直接補助は2.02兆円**
   - 大手17社への直接補助と、博報堂経由の中小事業者への補助の二重構造

5. **構造化JSONデータでは内訳が見えないが、費目・使途CSVには詳細がある**
   - `5-3_RS_2024_支出先_費目・使途.csv` に費目別の金額が記録されている
