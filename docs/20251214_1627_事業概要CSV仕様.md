## サンプルデータ

### 事業ID: 1 - 内閣人事局経費

```
事業の目的:
政府全体を通じた成果向上・人材育成を狙いとしており、全府省の職員に対する研修を行うことで、
政策の企画立案に係る能力・資質、国家公務員の職場において共通に必要な業務の管理に係る能力・
資質等を向上させることを目的としている。

現状・課題:
国家公務員採用試験の受験者の大幅な減少、若手職員の離職の増加が見られる一方、
行政ニーズの高度化・多様化が進む中、政府として公務を支える人材をいかに確保・育成していくかについて、
これまで以上に力を注ぐ必要がある。また、業務の効率的な実施や環境変化への対応、職員の人材育成や
能力の活用、組織文化の醸成等の観点から、幹部・管理職のマネジメントは極めて重要であり、
管理職員等のマネジメント力強化が課題となっている。

事業の概要:
1.新任幹部セミナー(本府省等の新任の審議官級を対象とする研修)
2.新任管理者マネジメント研修(本府省等の新任の管理職員を対象とする研修)
3.幹部候補育成課程中央研修(課長補佐及び係長を対象とする研修)
4.国家公務員合同初任研修(新規採用の係員等を対象とする研修)

事業区分: 前年度事業
事業開始年度: 2014
終了予定なし: TRUE
主要経費: その他の事項経費
実施方法: 直接実施
```

## 事業ビューでの活用方針

### 表示推奨項目（優先度★★★）

1. **事業の目的** (列14)
   - 事業の存在意義を説明
   - 折りたたみ可能なセクションで表示

2. **現状・課題** (列15)
   - 事業の背景を理解するために重要
   - 折りたたみ可能なセクションで表示

3. **事業の概要** (列16)
   - 具体的な実施内容
   - 箇条書きで表示（/で区切られている場合が多い）

4. **事業概要URL** (列17)
   - 外部リンクとして表示
   - 空の場合もあり（約30%程度）

### 補足情報（優先度★★）

5. **事業期間**
   - 事業開始年度 (列19) ～ 事業終了(予定)年度 (列21)
   - 「終了予定なし」の場合は「継続中」と表示

6. **事業区分** (列18)
   - 前年度事業/新規事業/等
   - バッジ表示推奨

7. **実施方法** (列25-30)
   - 複数選択可能（フラグの組み合わせ）
   - タグ表示推奨（例: 「直接実施」「補助」）

8. **主要経費** (列23)
   - 予算の性質を示す

## データ保存形式の提案

### 別JSONファイルとして保存

**ファイル**: `public/data/rs2024-project-details.json`

**構造**:
```typescript
{
  [projectId: string]: {
    projectName: string;
    purpose: string;          // 事業の目的
    currentIssues: string;    // 現状・課題
    overview: string;         // 事業の概要
    url: string | null;       // 事業概要URL
    category: string;         // 事業区分
    startYear: number | null; // 事業開始年度
    endYear: number | null;   // 事業終了(予定)年度
    noEndDate: boolean;       // 終了予定なし
    majorExpense: string;     // 主要経費
    implementationMethods: string[]; // 実施方法の配列
    ministry: string;         // 府省庁
    bureau: string;           // 局・庁
  }
}
```

### サイズ見積もり

- 1事業あたり平均: 約1-2KB（長文含む）
- 総サイズ: 約6-12MB（5,947事業）
- Gzip圧縮後: 約2-4MB（推定）

### 利用方法

1. **事業ビュー表示時**:
   - プロジェクト名でアクセス
   - `projectDetails[projectId]`でデータ取得

2. **部分読み込み**:
   - 必要に応じてAPI経由で個別フェッチも可能
   - 初回は全データをキャッシュ

## 実装時の注意点

1. **テキストの正規化**
   - `/`区切りは箇条書きに変換
   - 長文は折りたたみ表示
   - URL検証（一部空文字列あり）

2. **NULL処理**
   - 事業終了年度が空の場合の処理
   - URLが空の場合は非表示

3. **実施方法の配列化**
   - フラグ（1 or 空）を配列に変換
   - 表示名: 「直接実施」「補助」「負担」「交付」「分担金・拠出金」「その他」
