## Phase 2: レイアウト計算

グラフデータから事前計算済みレイアウト（座標 + Bezierパス）を生成する。
府省庁グリッド配置（6列×7行）で仮想空間を 2,600 × 8,261 px に収めた。

| タスク | 内容 | 状態 |
|--------|------|------|
| 2-1 | `scripts/compute-sankey2-layout.ts` 新規作成 | ✅ |
| 2-2 | グリッド配置（37府省庁を6列×7行、予算額降順） | ✅ |
| 2-3 | セル内X座標（type別ベースライン: ministry→project→recipient） | ✅ |
| 2-4 | セル内Y座標（金額比例高さ、線形スケール、GAP=0px） | ✅ |
| 2-5 | 支出先の府省庁帰属（最大フロー元の府省庁セルに配置） | ✅ |
| 2-6 | エッジのBezierパス事前計算（セル内 + クロスセル） | ✅ |
| 2-7 | レイアウト結果を `sankey2-layout.json.gz` に出力（5.4MB） | ✅ |

**完了条件**: 全37,258ノードにグリッド座標が割り当てられ、仮想空間 2,600 × 8,261 px。✅
