# 英字・カタカナのみ 未ラベル支出先の辞書化作業計画

## 目的

ラベリング辞書の作成者（自分）が、現時点で未ラベルの支出先のうち「外国法人・国際機関」に該当するものを効率的に特定・辞書化し、entity-labels-csv.json のカバレッジを向上させるため。

---

## スクリプト修正（2026-03-03 実施済み）

`analyze-label-coverage.py` の KAKU_PATTERNS が古いままで `generate-entity-labels-csv.py` と乖離していた。以下を同期した：

- KAKU_PATTERNS を全面更新（英語法人格サフィックス・個人・事業者・経費等の多数パターンを追加）
- `_HOMOGLYPH_TABLE` / `_normalize_for_match` を追加（キリル文字→ラテン文字正規化）
- `is_skip_target` に株式会社・NPO・独立行政法人等の早期リターン条件を追加
- 照合ループを `_normalize_for_match` 経由に変更
- `load_dictionaries` の `row.get('match_type', '')` が `None` を返すケースの NullPointerFix

**修正後の効果**:

| 指標 | 修正前 | 修正後 |
|---|---|---|
| 件数カバレッジ | 80.1% | **92.1%** |
| 未カバー件数 | 5,225件 | **2,065件** |
| 英字のみ未ラベル | 723件 | **402件** |
| カタカナのみ未ラベル | 95件 | **85件** |

## 作業フロー

### ステップ 1: リスト確認と分類判定（この会話内で実施）

1. 英字のみ 402件・カタカナのみ 85件を金額降順で上から順に確認
2. 各支出先に対して WEB 検索を実施し、以下の観点で分類：
   - **外国法人（外国企業）**: 外国籍の会社 → 辞書化不要（格パターンでカバーされるべき）
   - **外国政府機関**: 外国政府・州政府・省庁 → `foreign_agency_names.csv` に追加
   - **国際機関**: 国連機関・条約機関・多国間機関等 → `international_org_names.csv` に追加
   - **国内企業の海外法人**: 日本企業の現地法人 → 外国企業として格パターン対象
   - **日本法人の英語表記**: 日本の会社の英語名 → 辞書化対象外（`generate-entity-labels-csv.py` が正しく分類済み）
   - **判定不能**: 情報なし → スキップ

### ステップ 2: 追加パターン検討（KAKU_PATTERNS に追加すべきサフィックス）

調査を通じて新たに判明した法人格サフィックスは KAKU_PATTERNS に追加する：

- `\bAB\b` → スウェーデン・フィンランドの株式会社（Aktiebolag）→ `Dynasafe Demil Systems AB` 等
- その他判明次第随時追加

### ステップ 3: 辞書エントリ追加

分類結果に基づき、以下の辞書ファイルに追記する：

| 分類 | 辞書ファイル | L1 | L2 |
|---|---|---|---|
| 外国政府機関 | `public/data/dictionaries/foreign_agency_names.csv` | 外国法人・国際機関 | 外国政府機関 |
| 国際機関 | `public/data/dictionaries/international_org_names.csv` | 外国法人・国際機関 | 国際機関 |

#### foreign_agency_names.csv の書式

```
country_code,country_name,canonical_name,name,name_type
us,アメリカ合衆国,米空軍省,米空軍省,canonical
```

#### international_org_names.csv の書式

```
org_id,org_type,canonical_name,name,name_type
un,国連システム,国際連合,国際連合,canonical
```

### ステップ 4: ラベル再生成・確認

辞書追記・パターン追加後：

```bash
python3 scripts/analyze-label-coverage.py      # カバレッジ確認・uncovered_recipients.csv 更新
python3 scripts/generate-entity-labels-csv.py  # entity-labels-csv.json 再生成
```

## 既存パターンでカバー済み（辞書化不要）

- `(?i)\bInc\.?\b|\bCorp(?:oration)?\.?\b|\bLLC\b|\bLimited\b|\bLtd\.?\b|\bPty\.?\b|\bPLC\b`
- `\bB\.V\.|\bN\.V\.|\bGmbH\b|\bAG\b`
- `リミテッド$|インコーポレーテッド$|コーポレーション$`
- `(?<!\w)AS$`（ノルウェー・デンマーク Aksjeselskap）
