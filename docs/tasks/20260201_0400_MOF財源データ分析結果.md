### 3.3 公債金の内訳：新規国債と借換債

#### よくある誤解：「国債発行180兆円」

多くの報道で「日本は年間180兆円も国債を発行している」と言われますが、これは**誤解を招く表現**です。

#### 国債発行総額の真実

```
国債発行総額 約180兆円
├─ 借換債: 約145兆円（80%）
│  └─ 既存の国債の満期が来たので借り換え
│     → 新しい借金ではない
│     → 過去の借金の借り替え
│
└─ 新規国債: 約35兆円（20%）
   └─ これが真の「財政赤字」
      → MOFデータの公債金35.45兆円
```

#### MOFデータとの対応関係

| 項目 | MOFデータ（予算） | MOFデータ（決算） | 意味 |
|------|-----------------|-----------------|------|
| **歳入：公債金** | 35.45兆円 | 37.14兆円 | **新規国債**（真の財政赤字） |
| **歳出：国債整理基金への繰入** | 27.01兆円 | - | 国債の償還・利払い（借換債の原資） |

#### 借換債とは何か

**借換債（Refinancing Bond）**:
- 過去に発行した国債が満期を迎える
- その償還のために新たに国債を発行
- これは新しい借金ではなく、既存の借金の借り替え

**家計に例えると**:
```
住宅ローンの借り換え:
- 既存ローン残高: 2,000万円（A銀行）
- B銀行で2,000万円借り換え
  → 総借入額は変わらない
  → 「今年2,000万円も借金した」は誤解
  → 金利が下がるなどのメリットがある
```

#### 実際の財政状況

| 指標 | 金額 | 割合 |
|------|------|------|
| 歳入総額（当初予算） | 112.57兆円 | 100% |
| 租税収入 | 68.57兆円 | 61% |
| **新規国債（真の赤字）** | **35.45兆円** | **31%** |
| その他 | 8.55兆円 | 8% |

**財政赤字の規模**:
- 新規国債35.45兆円 ÷ 租税収入68.57兆円 = **約52%**
- つまり、税収の約半分を借金で賄っている状態

#### よくある誤解と正しい理解

| 誤解 | 正しい理解 |
|------|----------|
| ❌ 「日本は毎年180兆円も借金している」 | ✅ 新規の借金は約35兆円、残り145兆円は既存の借金の借り替え |
| ❌ 「国債発行額 > 予算 だから異常」 | ✅ 借換債を除くと予算内（新規国債35兆円 < 予算112兆円） |
| ❌ 「180兆円 ÷ 人口 = 一人あたり150万円の借金」 | ✅ 新規国債で計算すると一人あたり約28万円 |

#### なぜこの理解が重要か

1. **財政の実態把握**:
   - 真の財政赤字は35兆円
   - 180兆円という数字は実態を誇張している
   - ただし35兆円でも深刻な水準

2. **政策議論の質向上**:
   - 「国債発行180兆円」だけで批判するのは不正確
   - 本質的な問題は「新規国債35兆円」の削減

3. **国際比較の正確性**:
   - 他国との比較は「新規国債/GDP」で行うべき
   - 借換債を含めると正確な比較ができない

4. **財政健全化の議論**:
   - 削減すべきは「新規国債35兆円」
   - 借換債145兆円は削減の対象ではない
   - むしろ金利低下時には有利な条件で借り換え

#### 国債整理基金特別会計の役割

| 項目 | 金額 | 説明 |
|------|------|------|
| 一般会計からの繰入 | 27.01兆円 | 国債の償還・利払いのための資金 |
| 国債償還費 | - | 満期を迎えた国債の償還 |
| 利払費 | - | 国債の利子支払い |
| 借換債発行 | 約145兆円 | 償還した国債の借り換え |

**仕組み**:
```
一般会計
  ↓ 27.01兆円繰入
国債整理基金特別会計
  ↓ 国債償還
満期国債の償還
  ↓ 借換債発行（約145兆円）
新規国債発行（市場）
```

#### まとめ

✅ **正しい理解**:
- 国債発行総額: 約180兆円
  - うち新規国債: **約35兆円**（真の財政赤字）
  - うち借換債: 約145兆円（既存の借金の借り替え）

⚠️ **注意点**:
- 新規国債35兆円でも深刻な水準（税収の52%）
- 借換債があるということは、過去の累積債務が巨額であることを意味する
- 借換債を続けるには市場の信認が必要

### 11.3 決算データから得られた重要な知見

#### 1. 保険料収入が予想を上回った（+1.29兆円、+3.5%）

**要因**:
- **雇用情勢の改善**:
  - 完全失業率の低下
  - 正規雇用の増加
  - 厚生年金被保険者数の増加

- **賃金上昇**:
  - ベースアップの実施
  - 賞与の増加
  - 保険料収入は賃金に連動

- **物価上昇による年金額改定**:
  - 年金額が物価連動で増加
  - 保険料率も段階的に引き上げ

#### 2. 税金投入が予想を下回った（-1.89兆円、-12.0%）

**理由**:
- **保険料収入の増加による補填減**:
  - 保険料が予想より多く集まった
  - 税金からの補填が不要になった

- **年金給付の調整**:
  - マクロ経済スライドの適用
  - 給付額の伸びを抑制

- **執行の効率化**:
  - 不要な支出の削減
  - 予算の効率的な執行

#### 3. 年金財源の保険料依存度が上昇

| 指標 | 予算 | 決算 | 変化 |
|------|------|------|------|
| 保険料依存度 | 70% | **73%** | **+3pp** |
| 税金依存度 | 30% | **27%** | **-3pp** |

**意味**:
- より「自己負担」原則に近づいた
- 財政への負担が軽減
- 年金制度の持続可能性が向上

#### 4. 予算と実績の乖離から見る財政運営

**保守的な税収見積もり**:
- 租税収入: +8.2%の上振れ
- 安全マージンを持った予算編成
- 税収増は補正予算の財源に

**弾力的な財源配分**:
- 保険料収入増 → 税金繰入減
- 自動調整メカニズムが機能
- 財政の健全性向上
