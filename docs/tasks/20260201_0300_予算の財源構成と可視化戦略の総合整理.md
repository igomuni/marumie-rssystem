# 予算の財源構成と可視化戦略の総合整理

**作成日**: 2026年2月1日
**目的**: 一般会計・特別会計の財源構成を正しく理解し、誤解を生まない可視化戦略を策定する

---

## エグゼクティブサマリ

### 主な発見

1. **RSシステムのデータは「府省庁の予算総額」を示す**
   - 財源（税金 vs 社会保険料）の区別は含まれていない
   - 一般会計71兆円 + 特別会計77兆円 = 合計150兆円

2. **特別会計は1つの会計ではなく37種類の会計の総称**
   - 年金特別会計（68兆円）、労働保険特別会計（4兆円）など
   - それぞれが異なる財源を持つ

3. **特別会計にも税金が含まれる**
   - 特別会計77兆円のうち約14兆円は一般会計からの繰入（税金）
   - 残り約55兆円が社会保険料、約8兆円がその他財源

4. **一般会計の財源は税収と公債金**
   - 税収: 約62%（約44兆円）
   - 公債金（国債）: 約31%（約22兆円）
   - その他: 約7%（約5兆円）

5. **データ可視化は誤解を生むリスクがある**
   - 「厚生労働省92兆円」→「税金の無駄遣い」という誤解
   - 実際は約55兆円が社会保険料（国民が支払った年金保険料）

### 推奨アクション

1. **短期**: ツールチップに財源構成の注記を追加
2. **中期**: 特別会計の種類別表示を実装
3. **長期**: 財源別ビュー（税金/保険料/国債）を追加

## 1. 予算の全体像

### 1.1 予算の分類体系

```
国の予算（150兆円）
├─ 一般会計（71兆円）
│   財源: 税収62% + 公債金31% + その他7%
│
└─ 特別会計（77兆円）← これは「種別名」
    ├─ 年金特別会計（68兆円）
    │   財源: 年金保険料80% + 税金20%
    ├─ 労働保険特別会計（4兆円）
    │   財源: 労働保険料75% + 税金25%
    ├─ エネルギー対策特別会計（3.6兆円）
    │   財源: 石油石炭税 + 税金繰入
    └─ その他34種類（1.4兆円）
```

### 1.2 重要な理解：「特別会計」は種別名

**誤解しやすい理解**:
- 「特別会計」という1つの大きな財布がある

**正しい理解**:
- 「特別会計」は種別（カテゴリ名）
- その中に37種類の個別の特別会計が存在
- それぞれが異なる目的と財源を持つ

## 3. RSシステムのデータの性質

### 3.1 RSシステムが管理するもの

✅ **含まれている情報**:
- 歳出（支出）情報：どの事業にいくら使ったか
- 府省庁別の予算総額
- 事業別の予算・執行額
- 支出先情報

❌ **含まれていない情報**:
- 歳入（財源）情報：その財源が税収か公債金か保険料か
- 一般会計からの繰入の詳細金額
- 特別会計ごとの財源内訳

### 3.2 データの限界

#### 限界1: 財源情報がない

```
BudgetRecord {
  totalBudget: 28.37兆円  ← 総額のみ

  // 以下は含まれていない
  // taxRevenue: ?        ← 税収からいくら
  // bondRevenue: ?       ← 公債金からいくら
  // insuranceFee: ?      ← 保険料からいくら
}
```

#### 限界2: 一般会計からの繰入額が不明

年金特別会計の事業には：
- 総予算額: 28.37兆円（分かる）
- うち国庫負担: ?兆円（分からない）

#### 限界3: 金額の不一致

| 項目 | RSシステム | 財務省予算書 |
|------|-----------|-------------|
| 一般会計 | 71兆円 | 112兆円 |
| 差異 | 41兆円 | |

**理由（推測）**:
- RSシステムは執行ベース
- 財務省予算書は予算ベース
- 一部の会計が除外されている
- 集計方法の違い

## 4. データ可視化における誤解リスク

### 4.1 起こりうる誤解

#### 誤解1: 厚生労働省が税金を92兆円使っている

**誤解の内容**:
- サンキー図で「厚生労働省92兆円」を見る
- → 「厚生労働省が税金を92兆円も使っている！」

**現実**:
- 税金: 約37兆円（40%）
- 社会保険料: 約55兆円（60%）← **国民が支払った年金保険料**

**分断リスク**:
- 「年金が財政を圧迫している」
- 「厚生労働省は無駄遣いをしている」

#### 誤解2: 年金制度が財政を圧迫している

**誤解の内容**:
- 「年金に68兆円も使っている」
- → 「年金が財政を破綻させる」

**現実**:
- 年金給付68兆円のうち：
  - 保険料: 約54兆円（80%）← **自分たちが支払った**
  - 税金: 約14兆円（20%）← 不足分の補填

**正しい理解**:
- 年金は「保険制度」であり、主に保険料で賄われている
- 税金の投入は最低保障のため

#### 誤解3: 特別会計は不透明で無駄が多い

**誤解の内容**:
- 「特別会計は埋蔵金」
- 「不透明で無駄が多い」

**現実**:
- 年金特別会計は保険料を管理するための仕組み
- 一般会計と混ぜると保険料の使途が不明確になる
- 分離することで透明性を確保している

### 4.2 誤解を生む原因

#### 原因1: 財源の違いが見えない

現在のサンキー図:
```
予算総計（150兆円）
  ↓
厚生労働省（92兆円）← 見た目は巨大
  ↓
年金給付・医療等
```

**問題点**:
- 92兆円のうち何が税金で何が保険料か分からない
- 全てが「税金の無駄遣い」に見えてしまう

#### 原因2: 年金制度の仕組みへの理解不足

多くの国民が誤解していること:
- ❌ 「年金は税金で賄われている」
- ✅ 「年金は保険料で賄われ、不足分を税金で補填」

#### 原因3: 特別会計への不信感

過去の経緯:
- 特別会計は「埋蔵金」として批判された
- 「不透明」「無駄が多い」というイメージ
- しかし、年金特別会計は保険料管理の仕組みであり、本質的に異なる

## 7. データの見方ガイド（案）

### 7.1 一般会計と特別会計の違い

#### 一般会計（71兆円）

**目的**: 国の基本的な行政サービス

**財源**:
- 税収（所得税、法人税、消費税など）
- 公債金（国債発行）
- その他収入

**主な使途**:
- 社会保障（医療、福祉、生活保護など）
- 公共事業（道路、港湾、治水など）
- 文教及び科学振興
- 防衛
- 公債費（国債の利払い・償還）

#### 特別会計（77兆円）

**目的**: 特定の事業や資金を管理するため、一般会計と区分経理

**理由**:
1. 特定の収入で特定の事業を行う（受益者負担の原則）
   - 例: 年金保険料 → 年金給付
2. 一般会計と混ぜると使途が不明確になる
3. 事業の収支を明確にする

**主な種類**:
- 年金特別会計（68兆円）
- 労働保険特別会計（4兆円）
- エネルギー対策特別会計（3.6兆円）
- その他34種類（1.4兆円）

### 7.2 社会保険料とは

#### 定義

特定の社会保険制度を運営するために徴収される保険料

#### 主な種類

1. **年金保険料**
   - 厚生年金保険料（会社員・公務員）
   - 国民年金保険料（自営業者等）
   - 合計: 約54兆円

2. **労働保険料**
   - 雇用保険料
   - 労災保険料
   - 合計: 約1兆円

3. **医療保険料**
   - 健康保険料
   - 国民健康保険料
   - （RSシステムのデータには一部のみ含まれる）

#### 税金との違い

| 項目 | 社会保険料 | 税金 |
|------|-----------|------|
| 使途 | 特定の給付（年金、雇用保険など） | 一般的な行政サービス |
| 受益 | 支払った人が将来受け取る | 社会全体に還元 |
| 強制性 | 原則強制 | 強制 |
| 管理 | 特別会計 | 一般会計 |

### 7.3 年金制度の仕組み

#### 基本的な仕組み

```
現役世代
  ↓ 年金保険料を支払う
年金特別会計（68兆円）
  ├─ 保険料収入: 約54兆円（80%）
  └─ 国庫負担: 約14兆円（20%）← 税金
  ↓
高齢世代
  ↓ 年金給付を受け取る
```

#### なぜ国庫負担（税金）が必要か？

1. **少子高齢化**
   - 現役世代が減少 → 保険料収入が減少
   - 高齢世代が増加 → 給付額が増加

2. **最低保障**
   - 全国民に最低限の年金を保障する
   - そのために税金を投入

3. **法律で規定**
   - 国民年金法で基礎年金の1/2を国庫負担と明記

#### 世代間扶養

年金は「積立方式」ではなく「賦課方式」（世代間扶養）

```
現役世代（保険料を支払う）
  ↓
そのお金を使って
  ↓
高齢世代（年金を受け取る）
```

**重要**: 現役世代が支払った保険料は、自分のために積み立てられるのではなく、今の高齢世代を支えるために使われる

## 8. よくある質問（FAQ）

### Q1: なぜRSシステムに特別会計が含まれているのか？

**A**: 特別会計にも税金（一般会計からの繰入）が含まれているため

- 年金特別会計68兆円のうち約14兆円は税金
- 労働保険特別会計4兆円のうち約1兆円は税金
- 税金が使われている以上、行政事業レビューの対象

### Q2: 特別会計は本当に必要なのか？

**A**: 特定の収入で特定の事業を行う場合は必要

**理由**:
1. 受益者負担の原則
   - 年金保険料を支払った人が年金を受け取る
   - 保険料を一般会計と混ぜると使途が不明確になる

2. 事業の収支を明確にする
   - 年金事業の収支が明確になる
   - 透明性を確保

**注意**: すべての特別会計が必要というわけではなく、定期的な見直しは重要

### Q3: 厚生労働省の予算が92兆円もあるのはなぜ？

**A**: 社会保険料（年金保険料・労働保険料）が含まれているため

**内訳**:
- 一般会計（税金）: 23兆円
- 年金特別会計（主に保険料）: 68兆円
- 労働保険特別会計（主に保険料）: 1兆円

**重要**: 92兆円すべてが税金ではない。約55兆円は国民が支払った社会保険料

### Q4: 一般会計の約3割が国債というのは問題ないのか？

**A**: 財政健全化の観点から問題視されている

**現状**:
- 一般会計71兆円のうち約22兆円（31%）が公債金
- これは将来世代への負担転嫁を意味する

**財政健全化目標**:
- 政府は基礎的財政収支（プライマリーバランス）の黒字化を目指している
- しかし、達成は困難な状況

### Q5: このサンキー図で何が分かるのか？

**A**: 予算がどこに流れているかが分かる

**分かること**:
- どの府省庁にいくら予算が配分されているか
- その予算がどの事業に使われているか
- 最終的にどの支出先に支払われているか

**分からないこと**:
- 財源（税金 vs 社会保険料 vs 国債）の内訳
- 一般会計からの繰入の詳細金額
- 事業の効果や成果

### 9.1 データの正しい理解

1. **RSシステムのデータは「府省庁の予算総額」**
   - 財源（税金 vs 保険料）の区別は含まれていない

2. **特別会計は37種類の会計の総称**
   - それぞれが異なる目的と財源を持つ

3. **特別会計にも税金が含まれる**
   - 約18%は一般会計からの繰入（税金）

4. **一般会計の約3割は国債**
   - 将来世代への負担転嫁
