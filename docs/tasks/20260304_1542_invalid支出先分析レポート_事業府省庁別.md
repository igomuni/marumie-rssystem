# 問題あり支出先 深掘り分析レポート — 事業名・府省庁別

作成日: 2026-03-04
分析対象: `data/result/recipient_dictionary.csv` × `data/result/recipients_without_total.csv`

---

## 概要

invalid 率 30.7%（8,052件）の背景を、**事業名**・**府省庁**・**局庁**の軸でクロス集計した。
分析の結果、invalid の発生には**府省庁ごとの構造的な傾向**が確認され、改善優先度を明確に絞り込むことができた。

### 解説

#### 外務省（71.3%）— 構造的に最も深刻

在外公館・海外支援事業の性質上、支出先の多くが**外国の機関・個人・現地 NGO** であり、
日本の法人番号DBで照合できないことが根本原因。
単純な「入力ミス」ではなく、**制度的な対応策が必要な問題**。

- `大臣官房`（735件・64.0%）: 英字のみの外国機関名・個人名
- `国際協力局`（429件・74.0%）: ODA 先の途上国機関
- `領事局`（323件・77.1%）: 在外機関・任意団体
- `総合外交政策局`（319件・72.0%）: 国際機関・外国政府機関

#### 厚生労働省（25.0%）— 件数が最多・高額 invalid

件数 3,756 件・invalid 額 655,982億円（全府省庁中最大）。
職業安定局（ハローワーク）の入力ミスが最大要因。

- `職業安定局`（1,008件・72.4%・24,417億円 invalid）: 入力ミス 613件
  → ハローワーク系事業で事業所名・支店名の誤記入が多数

#### 農林水産省（34.9%）— 任意団体問題が構造的

農協・漁協・土地改良区の一部が任意団体として登録されているケース、
および補助金事業での複数農家・農業者集合表現が多い。

## 6. 改善提案（府省庁・事業別）

### A. 即改善可能（入力ルール周知で解消）

**対象: 厚生労働省 職業安定局 / 農林水産省 農産局・農村振興局**

- 失業等給付費等（19,524億 invalid）、水田活用の直接支払交付金（5,937億）、経営所得安定対策（4,130億）
- → **法人番号を入力して名称を自動補完するシステム**の義務化
- → 「支店・事務所名」「略称」での登録を禁止するバリデーション追加

### B. 制度設計が必要（外務省・在外機関）

外務省の invalid 率 71.3% は**外国法人・現地機関への支出という性質**から来るものであり、国内法人番号DBでの照合は原理的に不可能。

- → **外国機関用の別フォーマット**（登録国・法人種別・現地識別番号）を新設する
- → 外国の個人への支出（専門家費・奨学金等）は「個人受益者」として区別し、匿名化ルールを整備
- → JET プログラム等の外国青年個人は「個人招致受益者」として法人とは分離管理する

### C. 農水省の任意団体問題

農協・漁協の一部、地域農業法人以外の農業者団体が任意団体として登録されているケース。

- → 補助金交付の段階で**受取主体の法人番号を必須確認**する（任意団体の場合は構成法人を特定）
- → 任意団体への補助は「代表者名・法人番号（代表法人）＋任意団体フラグ」で管理

### D. 集合表現の廃止（環境省・世界自然遺産関連）

- 保全活動を複数団体に分配する場合、**各団体に個別に支出レコードを作成**する
- 集合記載が必要な場合は件数・金額の集計データとし、個別明細を別添で管理する

### E. 秘匿・匿名処理（法務省・公安調査庁）

調査委託先等で支出先を公開できないケースが構造的に存在する（知的財産権侵害対策等）。

- → **「秘匿」「非公開」フラグを設けた専用区分**を作り、`A`, `B`, `その他` 等の意味不明な記載を廃止する
- → 秘匿対象であることを明示した上で、金額・件数のみ公表するフォーマットを整備
