# データ可視化における誤解リスクと対策

**作成日**: 2026年2月1日
**背景**: サンキー図で厚生労働省92兆円という巨額が表示されることによる誤解・分断のリスク

---

## 1. 問題提起：データ可視化の両刃の剣

### 懸念事項

> 「一概に金額の規模だけ見ると年金が日本の財政の悪として分断を生みそうで憂いています」

この懸念は**極めて妥当**です。

### 起こりうる誤解

#### ❌ 誤解1: 厚生労働省が税金を92兆円使っている

**現実**:
- 厚生労働省92兆円のうち：
  - **税金**: 約37兆円（一般会計23兆円 + 特別会計への繰入14兆円）
  - **社会保険料**: 約55兆円（国民・企業が支払った年金保険料・労働保険料）

#### ❌ 誤解2: 年金制度が財政を圧迫している

**現実**:
- 年金給付69兆円のうち：
  - **保険料**: 約55兆円（80%）
  - **税金**: 約14兆円（20%）
- 年金は「保険制度」であり、主に保険料で賄われている

#### ❌ 誤解3: 特別会計は不透明で無駄が多い

**現実**:
- 年金特別会計は保険料を管理するための仕組み
- 一般会計と混ぜると保険料の使途が不明確になる
- 分離することで透明性を確保している

---

## 2. なぜ誤解が生まれるのか？

### 原因1: 財源の違いが見えない

現在のサンキー図：
```
予算総計（150兆円）
  ↓
厚生労働省（92兆円）← 見た目は巨大
  ↓
年金給付・医療等
```

**問題点**:
- 92兆円のうち何が税金で何が保険料か分からない
- 全てが「税金の無駄遣い」に見えてしまう

### 原因2: 年金制度の仕組みへの理解不足

多くの国民が誤解していること：
- ❌ 「年金は税金で賄われている」
- ✅ 「年金は保険料で賄われ、不足分を税金で補填」

### 原因3: 特別会計への不信感

過去の経緯：
- 特別会計は「埋蔵金」として批判された
- 「不透明」「無駄が多い」というイメージ
- しかし、年金特別会計は保険料管理の仕組みであり、本質的に異なる

## 3. データの正確な理解

### 厚生労働省92兆円の内訳（詳細版）

| 項目 | 金額 | 財源 | 説明 |
|------|------|------|------|
| **一般会計** | **23兆円** | **税金** | 社会保障、医療、労働政策等 |
| **特別会計（年金）** | 69兆円 | | |
| ├ 保険料収入 | 約55兆円 | 社会保険料 | 国民・企業が支払った年金保険料 |
| └ 国庫負担 | 約14兆円 | 税金 | 基礎年金の1/2等 |
| **合計** | **92兆円** | | 税金37兆円 + 保険料55兆円 |

### 税金の使途（正しい理解）

厚生労働省が税金から支出している金額：**約37兆円**
- 一般会計: 23兆円
  - 医療（国民健康保険への補助等）
  - 社会福祉（生活保護、障害者支援等）
  - 労働政策（職業訓練、雇用対策等）
- 特別会計への繰入: 14兆円
  - 基礎年金の国庫負担
  - 雇用保険の国庫負担

### 社会保険料の使途（正しい理解）

国民・企業が支払った保険料：**約55兆円**
- 年金保険料 → 年金給付（約55兆円）
- 労働保険料 → 雇用保険・労災保険給付（約3兆円）

**重要**: 保険料は「自分たちが将来受け取るための積立」

注: 基礎年金の1/2は法律で国庫負担と規定

## 5. 年金制度の正しい理解（参考）

### 年金は「保険制度」

| 項目 | 説明 |
|------|------|
| **保険料** | 現役世代が支払う |
| **給付** | 高齢世代が受け取る |
| **仕組み** | 世代間扶養（現役世代が高齢世代を支える） |
| **国庫負担** | 保険料だけでは不足するため、税金で補填 |

### なぜ国庫負担（税金）が必要なのか？

1. **少子高齢化**:
   - 現役世代が減少 → 保険料収入が減少
   - 高齢世代が増加 → 給付額が増加

2. **最低保障**:
   - 全国民に最低限の年金を保障する
   - そのために税金を投入

3. **法律で規定**:
   - 国民年金法で基礎年金の1/2を国庫負担と明記

### 年金給付69兆円の財源構成（推定）

| 財源 | 金額 | 割合 | 説明 |
|------|------|------|------|
| 厚生年金保険料 | 約40兆円 | 58% | 企業と従業員が折半 |
| 国民年金保険料 | 約15兆円 | 22% | 自営業者等が支払う |
| 国庫負担（税金） | 約14兆円 | 20% | 基礎年金の1/2 |
| **合計** | **約69兆円** | **100%** | |

**重要**: 年金給付の**80%は保険料**で賄われている

## 6. 「分断」を防ぐためのメッセージ

### ❌ 避けるべき表現

- 「厚生労働省が92兆円も使っている」
- 「年金が財政を圧迫している」
- 「特別会計は無駄の温床」

### ✅ 正確な表現

- 「厚生労働省の予算92兆円のうち、約55兆円は国民が支払った社会保険料です」
- 「年金給付の80%は保険料で賄われ、20%を税金で補填しています」
- 「年金特別会計は保険料を管理するための仕組みです」

### 伝えるべきポイント

1. **社会保険料は自分たちのお金**
   - 将来の自分が受け取るための積立
   - 他人のために使われているわけではない

2. **年金は世代間の助け合い**
   - 現役世代が高齢世代を支える
   - 自分も将来支えられる

3. **税金の投入は最低保障のため**
   - 保険料だけでは不足する部分を補填
   - 全国民に最低限の年金を保障

## 8. まとめ

### 問題の本質

データの可視化は**諸刃の剣**：
- ✅ 予算の流れを分かりやすく示す
- ⚠️ 誤解を生み、分断を助長する可能性

### 対策の核心

**透明性と正確性の両立**：
1. データを隠さない（透明性）
2. 正しく理解できる情報を提供する（正確性）
3. 誤解を生む表現を避ける（配慮）

### ユーザーへのメッセージ

このシステムの目的は：
- ❌ 特定の省庁や制度を批判すること
- ✅ 予算の流れを正しく理解すること
- ✅ より良い政策議論の基礎を提供すること

**「分断」ではなく「理解」を生むために**、
データの見せ方と説明に最大限の配慮を払う必要があります。
