## 1. 調査結果サマリー：151.1兆円の正体
`data/year_2024`に含まれるRSシステム（レビューシート）の集計データ（2023年度予算分）を分析した結果、指摘された **151.1兆円** という数値の正確な内訳を特定した。

| 会計区分 | 予算額（2023年度当初） | 備考 |
| :--- | :--- | :--- |
| **一般会計** | **72.56兆円** | RSに登録された事業のみの合計 |
| **特別会計** | **78.56兆円** | 年金特会、労働保険特会など |
| **合計** | **151.12兆円** | **RSシステムの支出合計と一致** |

## 2. 一般会計（MOF 114.4兆円 vs RS 72.56兆円）の乖離理由
財務省の一般会計予算（約114兆円）よりも、RSシステムの一般会計分（約72.6兆円）が少ない理由は、「レビューシート」の対象範囲に起因する。

RS（レビューシート）システムは主に「事業（プロジェクト）」を点検対象とするため、以下の項目のような「非事業的な支出」が一般会計の合計から除外されている。

## 4. 総合的な結論

### 4.1 RSシステムとMOFデータの対応関係

| 項目 | MOF総額（2023） | RS対象額（2023） | RS/MOF比率 |
|------|----------------|----------------|-----------|
| 一般会計 | 114.4兆円 | 72.56兆円 | 63.4% |
| 特別会計 | 441.9兆円 | 78.56兆円 | 17.8% |
| **合計** | **556.3兆円** | **151.1兆円** | **27.2%** |

**解釈**:
- RSシステムは国の予算全体（MOF歳出556兆円）の約27%をカバー
- 残りの73%は「事業レビュー対象外」の支出（国債費、地方交付税、給付金支払い、会計間繰入など）

---

### 4.2 RS対象外の主要項目（2023年度）

| カテゴリ | 主な内容 | 金額（兆円） |
|---------|---------|-------------|
| **国債関連** | 国債費（一般会計）+ 国債整理基金（特別会計） | 25.25 + 239.47 = **264.72** |
| **地方財源** | 地方交付税交付金 + 配付金特会 | 16.40 + 49.54 = **65.94** |
| **年金給付** | 年金特会（給付金支払い部分） | 約**50**（推定） |
| **財政投融資** | 融資・投資活動 | **24.94** |
| **予備費** | 新型コロナ対策等 | **5.50** |
| **その他** | 外国為替、特会繰入等 | 約**15** |
| **合計** | - | 約**426兆円** |

**残り（RS対象事業）**: 556 - 426 = **約130兆円**（実際のRS: 151兆円、補正予算含む）

---

### 4.3 結論

1. **RSシステムの151.1兆円は妥当な数値**である。これは「一般会計の事業分」と「特別会計の事業分」を合算したものである。

2. **MOFデータとRSシステムは補完的な関係**にある：
   - MOFデータ: 国の予算全体（556兆円）を網羅
   - RSシステム: 事業レビュー対象の事業予算（151兆円）に特化

3. **乖離の主要因が明確化**：
   - 国債関連（265兆円）: 国債費と国債整理基金
   - 地方財源（66兆円）: 地方交付税等
   - 年金給付等（50兆円）: 給付金支払いが中心
   - その他（45兆円）: 財政投融資、予備費、外為等

4. **データ活用の方向性**：
   - `data/year_2024` のRSデータ: 事業の詳細情報（支出先、契約内容等）を把握
   - MOFデータ: 予算全体の構造や財源構成を把握
   - 両者を組み合わせることで、国の予算の全体像を多角的に分析可能

## 付録D: データ抽出スクリプト

### 一般会計分析スクリプト

**スクリプト**: `scripts/analyze-mof-exclusions.ts`

このスクリプトは以下を実行します：
1. MOF一般会計歳出CSVファイルを読み込み
2. 「項名」カラムから国債費、地方交付税関連、予備費などを抽出
3. 年度別に集計して兆円単位で出力

**実行方法**:
```bash
npx tsx scripts/analyze-mof-exclusions.ts
```

### MOF特別会計分析スクリプト

**スクリプト**: `scripts/analyze-mof-special-accounts.ts`

このスクリプトは以下を実行します：
1. MOF特別会計歳出CSVファイルを読み込み（複数ファイル）
2. 特別会計別、勘定別に集計
3. RSシステムとの比較分析

**実行方法**:
```bash
npx tsx scripts/analyze-mof-special-accounts.ts
```

### RSシステム特別会計内訳分析スクリプト

**スクリプト**: `scripts/analyze-rs-special-accounts.ts`

このスクリプトは以下を実行します：
1. RSシステムの予算データ（2-2_RS_2024_予算・執行_予算種別・歳出予算項目.csv）を読み込み
2. 会計区分別、特別会計別、勘定別に集計
3. 年度間比較（2023 vs 2024）

**実行方法**:
```bash
npx tsx scripts/analyze-rs-special-accounts.ts
```
