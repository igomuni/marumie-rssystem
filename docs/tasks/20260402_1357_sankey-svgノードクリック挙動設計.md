# /sankey-svg ノードクリック挙動設計

## 目的

Google Mapsの操作感を参考に、ホバー・クリック・検索を一体化したノードインタラクションを設計する。現状はホバーのみで詳細確認でき、クリックは無反応。

---

## 参考: Google Mapsの操作モデル

| 操作 | 挙動 |
|------|------|
| アイコンホバー | ノード名をシンプルに表示する小さなツールチップ |
| アイコンクリック | 左サイドパネルに詳細情報（名称・評価・写真・住所など）が表示 |
| 検索 | 候補リストに表示 → 選択するとサイドパネルが開く |
| パネル外クリック | パネルが閉じる（地図はそのまま） |
| キャンバス非縮小 | パネルはキャンバスにオーバーレイ。地図（キャンバス）は縮まない |

**ポイント**: ホバーはシンプル（名前のみ）、クリックが「詳細を読む」アクション。検索と同じパネルを使い、操作の一貫性を保つ。

---

## 参考: /sankey2 のクリック挙動

| 挙動 | 詳細 |
|------|------|
| クリックで選択 | `selectedNodeId` 切替（再クリックで解除） |
| BFSハイライト | 選択ノードから上流・下流をBFS探索。距離に応じて opacity 減衰 |
| サイドパネル表示 | 左にパネルがオーバーレイ表示。キャンバス幅は変わらない |
| パネル内容 | ラベル・金額・ノード種別バッジ・流入元・流出先リスト |
| パネル内クリック | 接続ノードをクリックすると選択移動 + 自動ズーム |
| Escape / ✕ | パネルを閉じる |

---

## sankey-svg に固有の制約・文脈

- **ノード数が少ない**: 通常 50〜200ノード程度（sankey2 は数万ノード）
- **5列の固定構造**: total / ministry / project-budget / project-spending / recipient
- **ウィンドウ概念**: `recipientOffset` + `topRecipient` で表示範囲が変わる
- **集約ノード (`__agg-*`)**: 「その他 N件」扱い → ドリルダウン不可
- **キャンバスは全画面**: パネルはオーバーレイ。Sankey図の幅は常に全画面のまま

---

## 採用モデル: Google Maps型（ホバー軽量 + クリックでサイドパネル + 検索連動）

```
ホバー → ミニツールチップ（名前 + 金額のみ）
クリック → 左サイドパネルがSankey図にオーバーレイ表示（Sankey図は縮まない）
検索 → サイドパネルに候補表示 → 選択でノード詳細に切り替え
```

## 詳細仕様

### 1. ホバー（現行から変更）

**現状**: ホバーで詳細ツールチップ（名前・金額・接続先リスト）を表示  
**変更後**: シンプルなミニツールチップのみ（名前 + 金額の1行）

| 要素 | 内容 |
|------|------|
| 表示内容 | ノード名 + 金額（`formatYen`） |
| スタイル | 背景黒・角丸・影なし。現行より小さく |
| 位置 | マウスカーソルの右上（現行と同じ追随方式） |

> 接続先リストはホバーからサイドパネルに移動する。

---

### 2. クリック → サイドパネル

#### トリガー・解除

| 操作 | 挙動 |
|------|------|
| ノードをクリック | `selectedNode` を設定、左サイドパネル表示 |
| 再クリック（同ノード） | 選択解除、パネル非表示 |
| 別ノードをクリック | 選択を切り替え |
| SVG背景をクリック | 選択解除 |
| Escapeキー | 選択解除 |
| パネルの ✕ボタン | 選択解除 |

#### サイドパネルのレイアウト

```
┌──────────────┬─┐
│              │ │ ← 折りたたむボタン（右中央）
│ パネル内容   │◀│
│              │ │
└──────────────┴─┘
```

- `position: fixed; left: 0; top: 0; height: 100%` の固定パネル
- SVGコンテナは変更なし（パネルの裏に全画面のまま残る）
- パネル背景は白・不透明（`background: #fff`）
- パネル幅: `280px`（/sankey2 の 320px より狭め。情報が少ないため）
- 右辺の垂直中央に折りたたみボタンを配置（`position: absolute; right: -16px; top: 50%`）
  - 展開時: `◀`（chevron_left）、折りたたみ時: `▶`（chevron_right）
  - クリックでパネルを `width: 0` / `width: 280px` にトグル
  - 折りたたんだ状態でも `selectedNode` は維持（再展開で復元）

#### サイドパネルのコンテンツ

```
┌─────────────────────────────┐
│ [ノード名]              [✕] │  ← sticky header
│ 12.3兆円                    │
│ [府省庁] [財務省]           │  ← 種別バッジ + 府省庁名
├─────────────────────────────┤
│ 流入元                      │
│  • 予算総計  151.1兆        │
│  ...                        │
├─────────────────────────────┤
│ 流出先                      │
│  • 社会保険費  30.4兆       │
│  • 国債費      27.0兆       │
│  他 N件                     │
└─────────────────────────────┘
```

| 要素 | 詳細 |
|------|------|
| ノード名 | 改行あり（折り返し） |
| 金額 | `rawValue ?? value` を兆・億・円フォーマット |
| ノード種別バッジ | 総計 / 府省庁 / 事業（予算）/ 事業（支出）/ 支出先 |
| 府省庁名 | `node.ministry` がある場合のみ表示 |
| 集約バッジ | `aggregated: true` なら「集約 N件」を追加 |
| 流入元リスト | `sourceLinks` value降順・上位8件 + 「他N件」 |
| 流出先リスト | `targetLinks` value降順・上位8件 + 「他N件」 |
| リスト項目クリック | 接続ノードに選択を移動（ナビゲーション） |

**注意**: 集約ノード（`__agg-*`）のリスト項目はクリック不可（ノードが存在しないため）。

#### ハイライト挙動

`selectedNode` がある間：

| 対象 | opacity |
|------|---------|
| 選択ノード | 1.0 |
| 直接接続ノード（1ホップ） | 1.0 |
| それ以外のノード | 0.3 |
| 選択ノードに接続するリンク | 強調（現行のhoverと同等） |
| それ以外のリンク | 0.1 |

ホバー中はhoverハイライトが視覚的に優先（現行の動作を維持）。

---

### 3. 検索連動（将来対応）

Google Mapsと同様に、検索バーからノードを検索して選択できるようにする。

```
[検索バー] → 入力で候補リスト表示（インクリメンタルサーチ）
           → 候補クリック → selectedNode にセット → サイドパネルで詳細表示
           → ノードが画面内に収まるよう自動スクロール/ズーム
```

| 要素 | 詳細 |
|------|------|
| 対象 | 全ノード名（集約ノードを除く） |
| 検索方法 | 部分一致（前方一致優先） |
| 候補表示 | 最大20件、種別バッジ付き |
| 選択後 | サイドパネルで詳細表示 + 対象ノードをズームインして中央寄せ |

> 検索はサイドパネルと同じ「選択状態」を共有するため、一貫したUXになる。

## state の変更

| state | 内容 |
|-------|------|
| `selectedNode: LayoutNode \| null`（新規） | 選択ノード。`null` = 非選択 |
| 既存の `hoveredNode` | 変更なし |

---

## ホバー / クリック / 検索の優先関係まとめ

| 状態 | ツールチップ | サイドパネル | ハイライト |
|------|-------------|------------|-----------|
| 何も操作なし | 非表示 | 非表示 | なし |
| ホバー中 | ミニツールチップ表示 | 非表示 | hoverハイライト |
| クリック済み（非ホバー） | 非表示 | 表示 | selectedハイライト |
| クリック済み + ホバー | ミニツールチップ表示 | 表示 | hoverハイライト優先 |
| 検索で選択 | 非表示 | 表示 | selectedハイライト |
