# RSシステムとMOF歳入歳出データの対応関係分析：第1報（基礎年金勘定の調査）

## 1. 調査の背景と目的
MOF（財務省）の一般会計歳入（約114.4兆円）とRSシステムの支出合計（約151.1兆円）の間に生じている約36.7兆円の乖離の要因を特定するため、年金特別会計「基礎年金勘定」を中心に調査を実施した。

## 2. 調査結果サマリー（2023年度予算ベース）
MOFの予算データ（令和5年度）から抽出した主要な数値は以下の通り。

### 2.1 基礎年金勘定の規模
- **支出済歳出総額**: **28兆8,550億円**（`DL202378001b.csv`）
    - 内訳：
        - 基礎年金給付費：24兆6,945億円（約24.7兆円）
        - 国民年金勘定へ繰入：1,335億円
        - 厚生年金勘定へ繰入：1,799億円
        - その他諸支出金：約8.2億円
        - 翌年度繰越：3兆6,781億円

### 2.2 一般会計とのつながり
- **一般会計から年金特会への繰入額**: **約12兆2,000億円**
    - `DL202377001b.csv`（厚生労働省本省：基礎年金拠出金等年金特別会計へ繰入）
    - 基礎年金の約5割（国庫負担分）が一般会計から繰り入れられており、この分は一般会計の歳出合計（約127.6兆円※）に含まれている。
    - ※一般会計全体の歳出予算現額は約127.6兆円（`DL202377001b.csv`全行合計）。

### 2.3 乖離（約36.7兆円）の主な説明要因
一般会計に含まれない「保険料源泉の支出」が乖離の正体であると推測される。

1.  **基礎年金の保険料充当分**: **約16.65兆円**
    - 基礎年金勘定の支出現額（28.85兆円）－ 一般会計からの繰入（12.20兆円）
2.  **厚生年金保険給付費（報酬比例部分等）**: **約24.54兆円**
    - `厚生年金勘定`（`DL202378001b.csv`：保険給付費）の支出現額
3.  **合計**: **約41.19兆円**

これらの金額は、一般会計の歳入・歳出には現れないが、RSシステムが捕捉している「政府の活動全体」における支出には含まれている。年金だけで既に乖離額（36.7兆円）を上回る規模の支出が存在しており、RSシステムが特別会計を含む広範な予算・支出データを統合していることが数値的に裏付けられた。

## 3. その他の補足事項
- **年金生活者支援給付金給付費**: 2023年度で約5,242億円が執行されており、これは一般会計（厚生労働省本省）から直接支出される項目である。
- **2024年度（令和6年度）の推移**: 基礎年金給付費は30.04兆円に増加しており、今後も社会保障費の増大が乖離幅を広げる要因になると考えられる。

## 4. 今後の調査方針
- 年金以外の主要な特別会計（交付税及び譲与税配付金、国債整理基金等）における、一般会計との二重計上部分（繰入・繰出）を整理し、RSシステムの全体構造をより精緻に解明する。
