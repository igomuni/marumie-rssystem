# 支出先トップ50ベンダー・ロックイン調査報告 (再委託・事務局モデル精査版)

**調査日**: 2026年1月31日
**調査年度**: 令和6年度 行政事業レビューデータ（2023年度予算分）
**対象**: 民間企業および非営利法人受託額上位50社
**更新内容**: 調査フローに基づき、「事務局モデル」および「再委託構造（5-2データ）」の分析を追加。

## 1. 調査の全体概況

支出先トップ50を再精査した結果、単純な「特定企業への集中」だけでなく、受託者が実質的な業務を他社へ再委託する**「事務局・事業局モデル」**によるロックインが数多く確認されました。

### 主要な発見（再調査による追加）
- **事務局モデルの可視化**: 株式会社博報堂や大日本印刷株式会社などの案件において、5-2 CSVデータより具体的な再委託ルート（4系など）を確認しました。
- **事業局・代行型受託**: 地方整備局などの公的機関が受託しつつ、多数の外部企業へ再委託（4系以上）を行う「事実上の事務局」として機能しているケースを特定。
- **再委託のブラックボックス**: ITベンダー（富士通、NTTデータ等）の一部案件では、5-2データ上でも再委託先が公開されていない（なし判定）ものがあり、透明性に課題が残ります。

## 3. 「事務局・事業局モデル」の構造的課題

今回の再調査により、以下の構造が明らかになりました。

### 3.1 広告代理店・協会によるプラットフォーム受託
博報堂（4位）や大日本印刷（48位）、キャッシュレス推進協議会（21位）などが、**「資金分配・審査のプラットフォーム」**として機能し、そこから実務担当のITベンダーやコールセンター、印刷会社へ再委託される構造です。
- **リスク**: 事務局側の管理経費が巨額となり、実質的な支出効率が低下する可能性がある。

### 3.2 官公庁出先機関・独立行政法人による「事業集約」
地方整備局（19, 22, 25, 26, 29, 41, 47位にランクイン）が、道路事業などの巨大予算を一括で受け、そこから数多くの民間企業へ再委託（4系以上）を行うモデル。
- **分析**: これらは「事務局」的な役割を果たしており、再委託先を含めた全体構造が、特定の土木・ゼネコン等のロックインに繋がっていないかの監視が必要です。

### 3.3 再委託透明性の欠如
ハローワークシステム（富士通）のように、非常に高い占有率（80.7%）を持ちながら、5-2データ上で再委託先が「なし」とされているケースがあります。
- **考察**: これは実際に再委託がないのではなく、**「現行のRS開示基準では、特定の大手ITベンダー内部での受注分として処理され、その先の協力会社への資金の流れが捕捉できない」**という制度的限界を示唆しています。

## 4. 総括と次なる調査提言

今回の精査により、特定ベンダーへの依存は「受託した1社」で止まらず、その背後に複雑な**「再委託のネットワーク（n系）」**が存在することが確認できました。

特に「型」として**「事務局/BPR」**に分類された事業については、再委託先がどのように選定され、どの程度のマージンが発生しているかを、今後は5-3（費目・使途）データと組み合わせて深掘りする必要があります。
