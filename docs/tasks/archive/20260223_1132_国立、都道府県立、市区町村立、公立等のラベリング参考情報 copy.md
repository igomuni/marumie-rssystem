承知しました。
「国立・都道府県立・市区町村立・公立」等のラベリングは、**設置主体（設置者）による区分**です。
法人格とは別軸なので、制度的に整理します。

---

# ■ 1. 設置主体による基本分類

## ① 国立（こくりつ）

**設置者：国（中央政府）**

例：

* 東京大学
* 国立科学博物館

※実務上は
→ 多くが 国立大学法人 や独立行政法人により運営

---

## ② 都道府県立

**設置者：都道府県**

例：

* 東京都立大学
* 県立高校

法的には「地方公共団体設置施設」

---

## ③ 市区町村立

**設置者：市・区・町・村**

例：

* 市立病院
* 区立図書館
* 市立小学校

---

## ④ 公立（こうりつ）

**定義：地方公共団体が設置**

* 都道府県立＋市区町村立を包括
* 「国立」は通常含まない（文脈により含む場合あり）

例：

* 公立学校
* 公立病院

---

# ■ 2. 用語の関係整理

```text
設置主体区分
 ├─ 国立
 └─ 地方公共団体立
      ├─ 都道府県立
      └─ 市区町村立
```

※「公立」＝地方公共団体立の総称

---

# ■ 3. 私立との対比

| 区分 | 設置主体   |
| -- | ------ |
| 国立 | 国      |
| 公立 | 地方公共団体 |
| 私立 | 民間法人   |

例：

* 私立大学 → 学校法人
* 私立病院 → 医療法人など

---

# ■ 4. 注意点（制度的に重要）

### ① 設置主体と運営主体は異なる場合がある

例：

* 公立施設でも指定管理者制度で民間運営
* 国立大学は法人化済み（行政機関ではない）

### ② 「国営」という言葉は法的分類ではない

* 慣用表現
* 法令上は「国立」

---

# ■ 5. 分野別適用例

### 教育

* 国立大学
* 公立大学
* 私立大学

### 医療

* 国立病院機構病院
* 公立病院
* 私立病院

### 文化

* 国立美術館
* 県立美術館
* 私立美術館

---

# ■ 6. より精密な整理（法人格×設置主体マトリクス）

| 設置主体 | 行政機関 | 法人化主体         |
| ---- | ---- | ------------- |
| 国    | 省庁直轄 | 国立大学法人・独立行政法人 |
| 都道府県 | 直営   | 地方独立行政法人      |
| 市区町村 | 直営   | 地方独立行政法人      |
| 民間   | —    | 学校法人・医療法人等    |

---

# ■ 結論

* 「国立・公立・私立」は**設置者区分**
* 法人格とは独立概念
* 公立＝地方公共団体立
* 国立＝中央政府設置
