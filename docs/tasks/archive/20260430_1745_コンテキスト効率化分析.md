#### 2. CLAUDE.md に `/sankey-svg` の記載がない（中優先度）

**問題:**
- 最近の開発の主戦場：`app/sankey-svg/page.tsx`（最新の SVG Sankey UI）
- CLAUDE.md の Main Entry Points テーブルに記載なし

  - `app/lib/sankey-svg-*.ts` （filter, constants）

**現在の構造:**
```
Main Entry Points
├── /sankey      → page.tsx / api/route.ts / lib/sankey-generator.ts
├── /sankey2     → page.tsx / Sankey2View.tsx / scripts/generate-sankey2-data.ts
└── [/sankey-svg はなし] ❌
```
