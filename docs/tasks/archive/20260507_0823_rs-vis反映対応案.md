## 変更概要

今回の差分は `/sankey-svg` と `/subcontracts` のUI改善が中心。加えて Playwright 導入とエージェント/レビュー対応用の開発支援ファイルが追加されている。

rs-vis への反映では、公開アプリに必要なアプリ本体・ライブラリ・依存関係・圧縮済みデータを反映し、開発専用ファイルは除外する。

### 主な反映内容

- `/sankey-svg`
  - 一般会計・特別会計フィルタの追加。
  - ノードラベル常時表示と列間隔の自動調整。
  - ズーム時の列間隔拡大抑制、offsetボタン操作時のpan不具合修正。
  - 画面全体のフォントサイズ調整、Google Maps風の表示密度検討を踏まえたラベル/サイドパネル/検索結果表示の調整。
  - wheel zoom時のアニメーション停止、non-passive wheel listener化、列ラベルの全体表示時の重なり改善。
  - PlaywrightでのE2E/視覚確認シナリオ追加。
- `/subcontracts/[projectId]`
  - 再委託構造のカード型フロー・リスト型サイドパネル再設計。
  - 事業/府省庁ノード統合、縦方向フロー、ノードカードの構造・配色・ボーダー整理。
  - ラベルtruncate/hover tooltip、矢印ラベル重なり、複数直接支出ラベル重なり対策。
- データ生成/データ
  - `sankey-svg-2024-graph.json.gz`
  - `sankey-svg-2025-graph.json.gz`
  - `scripts/generate-sankey-svg-data.ts` 更新。
- 開発支援
  - Playwright設定・E2Eテスト追加。
  - CodeRabbit設定追加。
  - Agent skill / Claude command 追加。
  - ただし rs-vis への同期では `.agents/`, `tests/`, `playwright.config.ts` などは除外する。
