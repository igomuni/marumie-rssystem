# サイドパネル事業概要表示設計

## 目的

ユーザーが事業ノードを選択したときに、予算・支出額だけでなく事業の目的・概要などの質的情報をその場で確認できるようにするため。
また、RSシステム公式ページへのリンクを提供し、さらに詳しい情報へ素早くアクセスできるようにするため。

---

## 現状

- サイドパネルのヘッダー部分は「名称 / 予算額・支出額 / タグ」を表示
- その下に府省庁・事業・支出先の3タブがある
- 事業の質的情報（目的・概要・RSシステムリンク）は一切表示されていない

---

## 新機能スペック

### 表示条件

以下のノードを選択したときのみ表示する：

- `type === 'project-budget'` かつ `aggregated !== true`
- `type === 'project-spending'` かつ `aggregated !== true`

集約ノード（複数事業をまとめたもの）には表示しない。

### 表示位置

ヘッダー（名称・金額・タグ行）と3タブセクションの間に、アコーディオン形式で挿入する。

### アコーディオン仕様

- 初期状態: **折り畳み**
- 見出し行: `▶ 事業概要` / 展開時 `▼ 事業概要`（クリックでトグル）
- 見出し右端に RSシステムリンクアイコン（外部リンク）を常時表示
  - `https://rssystem.go.jp/project?q={encodeURIComponent(projectName)}&fiscalYear={year}`
  - `target="_blank" rel="noopener noreferrer"` で新しいタブで開く
  - ノード選択が変わっても展開/折り畳み状態はリセット（次の事業は初期折り畳み）

### 展開時に表示するフィールド

| フィールド | 型 | 表示名 |
|-----------|-----|--------|
| `overview` | string | 事業の概要 |
| `purpose` | string | 事業の目的 |
| `category` | string | 事業区分 |
| `startYear` / `endYear` | number \| null | 事業期間 |
| `implementationMethods` | string[] | 実施方法 |
| `url` | string \| null | 事業概要URL（あれば外部リンク表示） |

`overview` と `purpose` は長文なので全文表示。スクロール可能領域に収める（パネル全体の overflow: auto が既存で機能している）。

### データ取得

- APIエンドポイント: `GET /api/project-details/[projectId]?year=2024|2025`（このPRで year パラメータを追加）
- **取得タイミング**: ノード選択時に pre-fetch。折り畳み状態でもプレビューを即表示できるよう変更
- キャッシュキー: `"${year}-${projectId}"` で年度別に管理（年度切り替え時に再取得）
- ローディング中: `読み込み中...` テキスト表示
- 404 / エラー時 / サポート外年度（400）: `詳細情報が見つかりませんでした` 表示

### データ対応年度

このPRで `rs2025-project-details.json` を生成・追加した（6,061件）。

| year 状態 | project-details データ | 対応方針 |
|-----------|----------------------|---------|
| `'2024'` | `rs2024-project-details.json` | 表示可能 |
| `'2025'` | `rs2025-project-details.json`（本PRで追加） | 表示可能 |

## 実装設計

### 状態

`page.tsx` に追加：

```typescript
// アコーディオンの展開状態（selectedNode が変わったらリセット）
const [isProjectDetailExpanded, setIsProjectDetailExpanded] = useState(false);
// フェッチ済み詳細データ（projectId → ProjectDetail | null）
const [projectDetailCache, setProjectDetailCache] = useState<Map<number, ProjectDetail | null>>(new Map());
```

`selectedNodeId` 変更時（selectNode 呼び出し時）に `isProjectDetailExpanded = false` にリセット。

### フェッチロジック

ノード選択時（`useEffect([selectedNode?.id, year])`）：
1. project-budget / project-spending 非集約ノードのみ対象
2. キャッシュに `"${year}-${pid}"` キーが存在すれば skip
3. `fetch(/api/project-details/${pid}?year=${year})` を実行
4. 結果を `projectDetailCache` に保存（null も保存してリトライ抑止）

### JSX 配置

```text
ヘッダー div（名称・金額・タグ）
  ↓
アコーディオン div（project-budget / project-spending 非集約ノードのときのみ）
  - 見出し行（▶/▼ + "事業概要" + RSシステムリンクアイコン）
  - [折り畳み時] overview の3行プレビュー（スクロール可）
  - [展開時] 詳細コンテンツ全文
  ↓
3タブ div（府省庁 / 事業 / 支出先）
```
