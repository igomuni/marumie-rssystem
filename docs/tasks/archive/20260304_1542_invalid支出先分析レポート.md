## 1. 支店・事務所・支社名（法人単位への名寄せ未実施）

### 現状

法人の **支店・支社・事務所・地区本部** を独立した支出先名として登録している。法人番号DBは法人単位で登録されているため、突合不可となり「問題あり」と判定される。

**事例:**

| 記載名 | 法人番号のDB法人名 |
|--------|-----------------|
| (株)オキジム**北部支店** | 株式会社オキジム |
| (株)サーベイリサーチセンター**四国事務所** | 株式会社サーベイリサーチセンター |
| (株)建設技術研究所**北陸支社** | 株式会社建設技術研究所 |
| (株)ムトウ**苫小牧支店** | 株式会社ムトウ |

### 改善提案

- **支出先名は法人格を持つ主体（本社・法人）の名称で統一**し、支店・事務所名は記載しない
- 支店・事務所への支出も、法人格単位で集計・記載する
- 支払先の口座名義が支店名になる場合も、支出先名としては本社名称を使用する

---

## 2. 任意団体・共同体（法人番号なし）

### 現状

**1,102件**が法人番号なし・任意団体相当の支出先として記録されている。その内訳は大きく 2 種類に分類できる。

#### (a) JV・共同提案体・共同研究体（コンソーシアム）

複数法人が構成する共同事業体が支出先として記録されている。これらは法人格を持たないため、法人番号での識別が不可能。

**事例（抜粋）:**

```
建設技術研究所・建設技研インターナショナル・国際建設技術協会・八千代エンジニヤリング共同提案体
ドーコン・中央復建コンサルタンツ・パシフィックコンサルタンツ・日本交通計画協会共同提案体
BioAlchemy株式会社・三機工業株式会社・学校法人沖縄科学技術大学院大学共同研究体
リバーフロント研究所・日本生態系協会共同提案体
【代表】一般社団法人全国木材組合連合会【共同】一般財団法人日本木材総合情報センター、一般社団法人全日本木材市場連盟
```

#### (b) 協会・センター・協議会等（任意団体）

法人格（一社・公財等）がないにもかかわらず、実態は任意団体であるケース。

**事例（抜粋）:**

```
(一社)沖縄しまたて協会   → 法人番号DB未登録
(公財)資源環境センター   → 法人番号DB未登録
```

### 改善提案

#### JV・共同提案体について

- **代表法人と構成法人を分離して記録する**ことを原則とする
  - 代表法人（主たる契約相手方）の法人名と法人番号を「支出先名」とする
  - 構成法人は「備考」または別フィールドで補足記載
- やむを得ず共同体名を記載する場合は、法人番号フィールドに「代表法人の法人番号」を記載するルールを明示する

#### 任意団体について

- 法人格のない団体への支出は「法人番号なし（任意団体）」として明示的にフラグを立てる
- 可能な場合は実際の法的主体（構成法人または委任先法人）を特定して記載する

## 3. 入力ミス・名称不一致（CNあり、名称非一致）

### 現状

**957件**で、法人番号はデータに含まれているが、その法人番号に対応するDB上の法人名と支出先名が一致しない。単純な入力ミス・誤登録が多数を占める。

**事例（抜粋）:**

| 記載名 | 法人番号のDB法人名 | 推定原因 |
|--------|-----------------|---------|
| (株)ゼン**ン**リンデータコム | 株式会社ゼンリンデータコム | 誤字（ン重複） |
| (株)ナックイメージテクノロジ | 株式会社ナックイメージテクノロジ**ー** | 末尾の長音省略 |
| (株)**河井**工業 | 株式会社**河合**工業 | 誤字（異字体） |
| (株)**ノーパッド** | 株式会社**ノーパット** | 誤字（ッ→ッ/ト） |
| (株)エヌ・ティ・ティ・データ | 株式会社**NTT**データ | 表記揺れ（全角/略称） |
| (株)**日本通運** | **日本通運**株式会社 | 格の位置違い |
| (公財)京都産業21 | 公益財団法人にいがた産業創造機構 | 法人番号の誤登録 |

### 改善提案

- **法人名の入力時に法人番号DBとの整合確認を必須化**する（入力システムへの組み込み）
- 法人番号を入力すると法人名が自動補完される仕組みを導入する
- 「法人番号はあるが名称が不一致」のレコードに対する定期的なチェックバッチを設ける
- 表記ルール（全角/半角、格の位置など）を統一したガイドラインを整備する

## 4. 英字のみの支出先（表記ガイドライン未整備）

### 現状

**714件**が英字・数字・記号のみで構成された支出先名。これらは複数の性質が混在しており、一律に扱えない。

**事例（抜粋）:**

```
123Ink.ie           → 外国法人（アイルランド）
AEON(CAMBODIA)Co., Ltd  → 外国法人（カンボジア）
AGODA COMPANY PTE LTD   → 外国法人（シンガポール）
AAB VIET NAM CO,LTD.    → 外国法人（ベトナム）
AL CAR AG               → 外国法人（スイス）
A                       → 匿名・秘匿表記
AAA                     → 意味不明（プレースホルダー?）
AIT                     → 略称（国際熱帯工科大学?）
```

### 改善提案

- **外国法人の場合**、以下を追記することを必須とする：
  1. 登録国（例: `Agoda Company Pte. Ltd.（シンガポール）`）
  2. 法人形態の日本語説明（例: `LLC = 有限責任会社（米国）`）
- **略称のみの記載は禁止**し、正式名称を記載する（AIT → Asia Institute of Technology）
- `A`, `AAA` 等の意味不明な記載は入力時に拒否するバリデーションを設ける

---

## 5. 外国法人（国・種別の区別が困難）

### 現状

**22件**が外国法人の法人形態キーワード（LLC, Ltd, Inc, B.V., GmbH 等）を含む。ただし英字のみカテゴリと合わせると実態はより多い。

**事例（抜粋）:**

```
Bonnie Support Service Ltd（シドニー）
Elsevier B.V.ほか
GTT-EMEA Ltd
JB Line,Inc.（ボストン）
ASRC Energy Services Alasca,Inc.の再々委託先41社
QーBranch Services LLC
ワシントンコアLLC
Eーworld Energy & Water GmbH
```

### 改善提案

- **外国法人用の登録フォーマット**を新設する：
  ```
  法人名（原語）: [正式名称]
  法人名（カタカナ）: [カタカナ読み、任意]
  登録国: [国名]
  法人種別: [現地の法人形態 → 日本語対応表]
  ```
- 国税庁法人番号を持つ外国法人（日本支店設置済み）は法人番号で管理できる。持たない場合は「外国法人」フラグを明示する
- 再委託先（「〜の再々委託先41社」等）は支出先として記録せず、契約相手方のみ記載する

## 6. 集合・複数表現（複数エンティティの併記）

### 現状

**402件**で複数の法人や機関をまとめて1つの支出先名として記録している。

**事例（抜粋）:**

```
(株)NTTドコモ,株式会社筑水キャニコム,阿蘇森林組合        → カンマ区切り並列
ENEOS株式会社ほか                                      → 「ほか」付き
SBテクノロジー株式会社・日本電気株式会社共同提案体         → 中点区切り
【代表】一般社団法人全国木材組合連合会【共同】一般財団法人日本木材総合情報センター、...
UNDP、ICAO等の派遣先国際機関
```

### 改善提案

- **支出先名の記載は1レコード1法人を原則**とする
- 複数法人への分配支出は行を分けて記録する
- 「ほか」「など」「等」を支出先名に含めることを禁止する
- システム側で中点（・）や読点（、）区切りの支出先名を警告表示する

## 7. 「その他」プレースホルダー乱用

### 現状

**118件**で「その他」を支出先名として使用。支出の宛先が特定できない。

**事例（抜粋）:**

```
その他
その他(251社)
その他(37事業者)
その他(71自治会)
その他(37局)
その他(不用額)    ← 支出先でなく不用額の記録
```

### 改善提案

- `その他` 単独での支出先記載を**原則禁止**とする
- やむを得ない場合（少額多数の集計等）は専用フィールドで集計件数・総額を記録し、個別の支出先名は別添明細で補完する
- `その他(不用額)` 等、支出でないものは別区分で管理する

---

## 8. 事業・プログラム名が支出先に記録されているケース

### 現状

**56件**で、支出先（法人・機関）ではなく **事業名・基金名・プロジェクト名** が支出先フィールドに入力されている。

**事例（抜粋）:**

```
GPE基金
IPPF日本信託基金
アマゾン基金
アジア開発銀行信託基金
コロンボ計画
SAGAスマートアイランドプロジェクト
ゲノム・データ基盤プロジェクト
みつばちプロジェクト
```

### 問題点

支出の実際の受取主体（法人・機関）が不明になる。また、事業名は改廃・統合が多く、法人番号による継続追跡ができない。

### 改善提案

- **支出先は必ず法人または行政機関の名称**で記録する
  - 基金への支出 → 基金の管理主体法人（例: アジア開発銀行）を記載
  - 計画・プロジェクトへの支出 → 事務局・受託法人を記載
- 事業名を補足したい場合は「支出先備考」フィールドを設ける

---

## 9. 法人格あり・法人番号未登録（493件）

### 現状

名称に `株式会社` 等の法人格が含まれているにもかかわらず、法人番号DBに存在しないケース。主な原因は以下の通り。

**事例（抜粋）:**

```
8PEAKS RESORT株式会社       → 新設法人（DB未収録の可能性）
CYBERDYNE株式会社           → 法人番号と名称が不一致の可能性
A株式会社, B株式会社        → 匿名・秘匿処理された法人名
A漁業協同組合, B漁業協同組合 → 同上
株式会社〇〇・△△株式会社共同提案体 → JV形式で名称に法人格が混在
```

### 改善提案

- **法人番号は必須入力項目**とし、未登録の場合は入力不可にするか、事前申請フローを経た上で記録する
- 新設法人等でDB未収録の場合は「法人番号申請中」のステータスフラグを設ける
- 匿名・秘匿処理が必要な場合は、別途非公開区分として管理し、公開データとは分離する
