## 結論

この4事業は、通常の「府省庁 → 直接支出ブロック → 再委託ブロック」という見方だけでは足りない。

5-2 の明示的ルート、つまり `支出元の支出先ブロック` が空の行は、いずれも府省庁起点である。  
ただし、5-2 のブロック接続グラフ上では、府省庁から直接支出されたブロックとは別に、以下のような「別起点ブロック」が支出元として現れる。

- 財政融資資金借入金
- 回収金等
- 自己収入等
- 利水者等
- 市中金融機関
- 独立行政法人、国立研究開発法人などの法人自身

これらは `支出元の支出先ブロック名` が空なのではない。むしろ、ブロック名は入っている。  
問題は、そのブロックが「どこから入ってきた資金か」を 5-2 の通常ルートだけでは説明できない点にある。

したがって、UIでは次の区別が必要。

- **明示ルート**: 5-2 で `支出元の支出先ブロック` が空、`担当組織からの支出=TRUE`
- **別起点ブロック**: 支出元として現れるが、担当組織からの直接支出ブロックではない
- **ブロック内訳**: 5-1 の合計支出額、支出先数、役割
- **費目・使途**: 5-3 にある場合だけ詳細として表示

## UIへの示唆

### 1. ルートは1種類ではない

明示的な府省庁ルートだけでなく、別起点ブロックを表示できる必要がある。

分類案:

| 分類 | 判定例 | 表示 |
|---|---|---|
| 府省庁起点 | `支出元ブロック` 空、`担当組織からの支出=TRUE` | 通常のルート |
| 別財源/別起点 | 支出元として現れるが、直接支出ブロックでない | 別起点として表示 |
| 移替 | 補足情報に `移替` | 国内部の移替として表示 |
| 自己収入/借入/回収金 | ブロック名や役割に `自己収入`, `借入`, `回収金` | 公費とは別色/タグ |
| 参考的制度フロー | 金額ゼロ、支出先内訳なし、補足に `融資`, `政府保証借入` など | 参考フローとして弱く表示 |

### 2. フロー図とリストは分担する

フロー図は全構造を詰め込むより、次の関係だけを示すのがよい。

- 府省庁起点
- 別起点
- 合流先ブロック
- 下流ブロック

詳細は `資金の流れリスト` と `ブロック一覧` で読む。

### 3. ブロック一覧の主要列

この4事業を見る限り、最低限ほしい列は次。

- ブロックID
- ブロック名
- 起点種別
- 合計支出額
- 支出先の数
- 事業を行う上での役割
- 補足情報
- 費目・使途の有無

### 4. 金額ゼロの制度フローを通常支出と分ける

PID 19873 のように、5-2 には流れがあるが 5-1/5-3 に金額・支出先内訳がない事業がある。

このタイプは、通常の支出フローと同じ意味で面積や太さを持たせるべきではない。  
`制度フロー`, `参考フロー`, `金額内訳なし` のようなタグが必要。

## 類似ケース洗い出し

### 判定条件

2025年データで、次を満たすブロックを候補として抽出した。

1. 5-2 で `支出元の支出先ブロック` として登場する
2. そのブロックが、5-2 の `支出先の支出先ブロック` に登場しない
3. そのブロックが、担当組織からの直接支出ブロックでもない
4. 5-1 に `支出先名` と `支出先の合計支出額` がある

この条件は、「5-2上は別起点のように見えるが、sankey-svg上は支出先として集計されうるブロック」を拾うためのもの。

広めに拾うと、2025年では **24事業 / 26ブロック / 約3.46兆円** が該当した。  
この中には、都道府県や管区警察局など、実施主体・経由主体として支出先表示しても完全には誤りと言い切れないものも含まれる。

### 広めの候補に含まれる別タイプ

広めの候補には、財源というより「事業実施上の経由主体」や「参考的な実施形態」に見えるものもある。

例:

- PID 2776 `障害者自立支援給付`: `都道府県 -> 市区町村`
- PID 3279 `畜産・酪農収益力強化総合対策基金等事業`: `都道府県 -> 協議会`
- PID 333/334/335/336/359/6447 警察庁系: `関東管区警察局ほか` 等から民間会社へ
- PID 934 `地域人権問題に対する人権啓発活動の委託`: `地方公共団体(88団体) -> 長野県`、補足 `実施形態の一例`

これらは「支出先に出るのが常に誤り」とは言い切れない。  
ただし `sankey-svg` の通常の支出先列に混ぜると、府省庁からの支出先なのか、別起点/経由主体なのかが分からなくなる。

### UI上の扱い

`sankey-svg` では、少なくとも次を区別できるとよい。

- 通常の支出先
- 別財源・自己収入・借入・回収金
- 移替
- 事業実施上の経由主体
- 参考的な実施形態

PID 1406 の `財政投融資資金借入金` と `回収金等` は、通常の支出先ではなく、`別財源/回収金` として扱うべき。
