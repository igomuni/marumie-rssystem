# 未ラベリング改善設計

## 目的

支出先ラベリング画面を見たデータ分析者が、1896件・1918億円に及ぶ未ラベリング支出先の正体を把握できるようにするため。また全体カバレッジを向上させ、「なぜラベルが付いていないのか」の原因別対処を明確にする。

## 原因別の設計

### 1. 事業名混入（~714億円）

**問題**: 支出先名に「事業を受けている実施機関名」ではなく「事業名・プロジェクト名」が入っている。原データの記入ミスに起因する。

**対象パターン**:
- 研究プロジェクト系: `〜の構築`、`〜プロジェクト`、`〜エコシステムの構築`
- 委託業務系: `〜業務`、`〜製表事業`、`〜推進事業`、`〜に関する業務`

**設計案**:
- KAKU_PATTERNS に新L2 `事業名` を追加（L1=`その他`）
- 既存の `プレースホルダー` L2 を拡張するのではなく、別L2 `事業名` を新設して識別可能にする
- 「〜の構築」は内閣府 BRIDGE プログラム（SIP等）の研究課題名が入っており、実支出先は別途記録されているケースが多い

### 2. 外国法人（英字）（~156億円）

**問題**: 既存の KAKU_PATTERNS は特定サフィックス（`Inc.`, `Corp.`, `GmbH` 等）に依存しており、サフィックスなし英字名称（`Orano Recyclage`, `Kongsberg Satellite Services`等）が取りこぼされている。

**設計案**:
- **辞書追加**: 金額上位エントリを `foreign_agency_names.csv` または新規 CSV に明示的に追加
- **パターン追加**: 英字のみで構成される名称（カタカナなし・漢字なし）を包括的にキャッチするパターン
  - ただし単一英字 A〜J は別カテゴリ（匿名化）なので除外

### 3. 地方公共団体集合（~109億円）

**問題**: 「15市町村等」「外12府県」「その他(概算払い1県・1市含む)」等のパターンが aggregate_names.csv に未収録。

**設計案**:
- `aggregate_names.csv` にパターン追加:
  - `\d+市町村等` → `地方公共団体集合`
  - `外\d+府県` → `地方公共団体集合`
  - `その他\(\d+団体\)` → `地方公共団体集合`
  - `その他\(\d+県\)` → `地方公共団体集合`
  - `滋賀県その他`、`群馬県他\d+県` 等の個別名称 → exact match
  - 「市区町村(約N市町村)」形式 → regex

### 4. 国際機関（未認識）（~79億円）

**問題**: 「世界貿易機関（WTO）」「世界気象機関（WMO）」「化学兵器禁止機関（OPCW）」等が `international_org_names.csv` または KAKU_PATTERNS に未収録。

**設計案**:
- `international_org_names.csv` に追加（exact match）:
  - 世界貿易機関、化学兵器禁止機関、世界気象機関、関税協力理事会、アジア生産性機構、ユネスコ、アマゾン基金、教育を後回しにはできない基金、GPE基金、カーネギー国際平和財団政策プログラム、国際戦略研究所アジア(IISS)、経済協力開発機構原子力機関 等
- KAKU_PATTERNS: `外国法人・国際機関/国際機関` パターン拡張（国際機関の日本語名称パターン）

### 5. 匿名化エントリ（~96億円）

**問題**: 単一英字（A, B, C...J）や「地方公共団体1」「地方公共団体2」等の匿名化された名称が未ラベリング。

**設計案**:
- KAKU_PATTERNS に追加:
  - `^[A-Z]{1,2}$` → `その他/プレースホルダー`（匿名化された企業・団体）
  - `^地方公共団体\d+$` → `地方公共団体/地方公共団体集合`

### 6. 経費・費目名（~63億円）

**問題**: 「法人共通」「審査業務費」「研究業務費」「事務費」「積立・繰越金」等が費目名称として記入されており、支出先名ではない。

**設計案**:
- KAKU_PATTERNS の `プレースホルダー` L2 を拡張:
  - `^法人共通$`、`^法人$`、`^審査業務費$`、`^研究業務費$`、`^事務費$`、`^積立・繰越金$`
  - `調査費$`、`委託費$`、`補助金$`（単体で支出先名になっているもの）

### 7. 国際機関事務局（~17億円）

**問題**: APEC・OAS・SAARC 等の国際機関の事務局が未分類。

**設計案**:
- KAKU_PATTERNS に追加:
  - `(APEC|OAS|PIF|SAARC|GEO|MOPAN|APG|PECC|ABAC|RCEP|カリコム|日中韓|国際獣疫|国際穀物|博覧会国際).*事務局$` → `外国法人・国際機関/国際機関`

### 8. 受益者集合（~24億円）

**問題**: 「文化功労者」「申請人」「通訳人」「沿岸漁業者」がKAKU_PATTERNSに未収録。

**設計案**:
- KAKU_PATTERNS に追加:
  - `^文化功労者$` → `その他/受益者集合`
  - `^申請人$` → `その他/受益者集合`
  - `^通訳人$` → `個人/通訳者`（個人として扱うか受益者集合か要確認）
  - 「沿岸漁業者」は既存パターン `^その他.*漁業者` の厳密化が必要（`^漁業者` 等）

### 9. 国内事務局（~90億円）

**問題**: 「全国旅行支援群馬県事務局」（878億円）が突出して大きい。これは2022年度の全国旅行支援の執行機関。

**設計案**:
- 「全国旅行支援群馬県事務局」→ `地方公共団体/市区町村` または `その他/事務局` として分類
  - 実態は都道府県が設置した事務局なので `地方公共団体` が妥当
- 一般的な `〜事務局` パターン:
  - 固有名 → 個別辞書登録
  - 「カルテット事務局」等の文化芸術系 → `その他/実行委員会等`

## 非対象事項

- 「その他（分類困難）」~527億円の個別対応（費用対効果が低い）
- 外国法人の包括的パターン追加（誤検知リスクが高く別タスクとして検討）
- 公益法人・NPO系の未認識（各法人格サフィックスのない名称への対応は別タスク）
