# `その他(集合)` ラベルの `集合` / `匿名` / `プレースホルダー` 再分類設計

## 目的

データ分析者が支出先ラベルを見たとき、「複数エンティティをまとめた集合表現なのか」「匿名化された個別受益者なのか」を区別できるようにする。現在の `その他(集合)` というラベルではこの区別ができず、分析時の解釈が曖昧になっている。

---

## 変更方針

### L1 の変更

`その他(集合)` → `その他`

`(集合)` サフィックスは、下記の L2 の再分類によって「集合か匿名か」の情報を L2 で表現するため不要になる。

### L2 の再分類

| 現在 | 変更後 | 意味 |
|------|--------|------|
| `集合名称`（地理・行政カテゴリ） | `{カテゴリ}集合`（後述） | 複数エンティティをまとめた名称 |
| `集合名称`（特定カテゴリ不明） | `集合名称`（維持） | 法人種別が混在する集合表現 |
| `受益者` | `匿名受益者` | 匿名化された給付対象者 |
| `プレースホルダー` | `プレースホルダー`（維持） | 汎用プレースホルダー |

---

## 具体的な変更内容

### 1. KAKU_PATTERNS の変更

全エントリの L1 `その他(集合)` を `その他` に変更。L2 は以下の方針：

| 現在 | 変更後 |
|------|--------|
| `('その他(集合)', '集合名称', r'^病院、訪問看護ステーション等$')` | `('その他', '集合名称', ...)` |
| `('その他(集合)', '集合名称', r'^他[0-9]+')` | `('その他', '集合名称', ...)` |
| `('その他(集合)', '集合名称', r'支出先上位\d+者以降...')` | `('その他', '集合名称', ...)` |
| `('その他(集合)', '集合名称', r'^その他\d+局$')` | `('その他', '行政機関集合', ...)` |
| `('その他(集合)', '集合名称', r'^その他.*センター連合...')` | `('その他', '集合名称', ...)` |
| `('その他(集合)', 'プレースホルダー', r'^その他$...')` | `('その他', 'プレースホルダー', ...)` |
| `('その他(集合)', '受益者', r'^利水者...')` | `('その他', '匿名受益者', ...)` |
| `('その他(集合)', '受益者', r'^ハンセン病療養所...')` | `('その他', '匿名受益者', ...)` |

`その他\d+局$` は「局」が行政機関を示すため L2 を `行政機関集合` に変更。

### 2. DICT_LABEL / 辞書読み込みの変更

#### `beneficiary_names.csv`

```python
'beneficiary_names.csv': ('その他', '匿名受益者'),
```

シンプルに L1/L2 を更新（全行一律）。

#### `aggregate_names.csv`

現在は全行に `('その他(集合)', '集合名称')` を固定割り当てしているが、**`category` フィールドに応じて L2 を変える** 必要がある。

`DICT_LABEL` の値を `None` とし、辞書読み込みロジックに専用ブランチを追加する（または辞書読み込み関数を `category` 対応に拡張する）。

**category → (L1, L2) マッピング:**

| category | L1 | L2 |
|---|---|---|
| `都道府県` | `その他` | `都道府県集合` |
| `市区町村` | `その他` | `市区町村集合` |
| `地方公共団体` | `その他` | `地方公共団体集合` |
| `行政機関` | `その他` | `行政機関集合` |
| `国際機関` | `その他` | `国際機関集合` |
| `受益者` | `その他` | `匿名受益者` |

`beneficiary_agg`（中小企業者）は category が `受益者` のため `匿名受益者` に分類する。中小企業者が補助対象として記載される文脈と一致する。

## 変更後のラベル体系（`その他` L1 内）

| L2 | 説明 | 件数感 |
|----|----|--------|
| `都道府県集合` | 複数都道府県の集合 | 辞書登録あり |
| `市区町村集合` | 複数市区町村の集合 | 辞書登録あり |
| `地方公共団体集合` | 都道府県+市区町村混合 | 辞書登録あり |
| `行政機関集合` | 複数行政機関の集合 | 辞書登録 + KAKUパターン |
| `国際機関集合` | 複数国際機関の集合 | 辞書登録あり |
| `集合名称` | 法人種別混在の集合表現（「病院・訪問看護ステーション等」等） | KAKUパターン |
| `匿名受益者` | 匿名化された個人・受益者カテゴリ | 辞書登録 + KAKUパターン |
| `プレースホルダー` | 汎用プレースホルダー（「その他」等） | KAKUパターン |
