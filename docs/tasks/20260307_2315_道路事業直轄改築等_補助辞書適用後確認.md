### 残存invalidの評価

補助辞書適用後に残る50件のinvalidは、**構造的にカバー困難なもの**に収れんした:

- `その他` 12件 → 真の不透明支出先（軸5でも正しく減点）
- `個人(X)` 20件 → 道路用地取得の正当な匿名化（設計判断待ちのカテゴリB）
- JV・設計共同体 18件 → プロジェクト固有の一時的共同体（正規表現対応が必要なカテゴリD）

**正当invalid（その他）の割合**: 12/138 = **8.7%** → 軸5スコア82.6点と完全整合。

### 軸2が低い理由

法人番号記入率63.8%（50行が未記入）。調査が必要な点:
- 地方整備局のCNは `2000012100001`（国土交通省）として登録されており、このCNは実際に記入されているか？
- 個人(X)・その他・JV のCNはそもそも存在しない → 構造的に100%は達成困難
