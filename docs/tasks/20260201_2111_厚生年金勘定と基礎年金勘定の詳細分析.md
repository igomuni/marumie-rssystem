# 厚生年金勘定と基礎年金勘定の詳細分析：RSシステムとMOFデータの乖離解明

## 調査の背景

RSシステムとMOFデータの年金特別会計において、以下の乖離が確認されていた：

- **厚生年金勘定**: RS 25.01兆円 vs MOF 50.41兆円（差額25.40兆円、50.4%）
- **基礎年金勘定**: RS 28.37兆円 vs MOF 28.85兆円（差額0.48兆円、1.7%）

この25.40兆円の差額の内訳を明らかにするため、MOFデータおよび厚生労働省の公式資料を詳細に分析した。

---

## 4. 結論

### 4.1 厚生年金勘定の差額25.40兆円の内訳

| カテゴリ | 金額（兆円） | 構成比 | 説明 |
|---------|-------------|--------|------|
| **基礎年金勘定への繰入** | **20.33** | **80.0%** | 厚生年金から基礎年金への財源移転 |
| **実施機関交付金** | **4.76** | **18.7%** | 国家公務員共済組合等への交付金 |
| 業務勘定への繰入 | 0.22 | 0.9% | 年金相談事業費等 |
| 予備費 | 0.10 | 0.4% | 予備費 |
| **合計** | **25.40** | **100%** | - |

**主要因**：
- **80%は基礎年金勘定への繰入**であり、これは会計間の財源移転
- **19%は共済組合への交付金**であり、実施機関への交付
- これらは「事業レビュー」の対象ではなく、制度間の財源調整・移転

### 4.2 RSシステムとMOFデータの役割分担

| データソース | 対象範囲 | 金額 |
|-------------|---------|------|
| **RSシステム** | 事業レビュー対象の「保険給付費」のみ | 25.01兆円 |
| **MOFデータ** | 会計全体（給付 + 会計間繰入 + 交付金等） | 50.41兆円 |

**解釈**：
- RSシステムは「実際の年金給付」（24.99兆円）に特化
- MOFデータは会計全体の資金の流れ（50.41兆円）を網羅
- 差額25.40兆円は、主に「会計間の財源移転」（20.33兆円）と「共済組合交付金」（4.76兆円）

### 4.3 基礎年金勘定はほぼ完全一致

- RS: 28.37兆円（基礎年金給付費のみ）
- MOF: 28.85兆円（給付費 + 他勘定繰入 + 予備費）
- **差額わずか0.48兆円（1.7%）**

基礎年金勘定は「給付金の支払い」が中心であり、RSシステムとMOFデータがほぼ一致する。

## 5. 財源の流れ（年金特別会計全体）

```
【一般会計】
  ↓ 基礎年金拠出金（12.48兆円）
【基礎年金勘定】
  ・基礎年金給付費: 28.37兆円
  ↑ 厚生年金勘定から繰入（20.33兆円）
  ↑ 国民年金勘定から繰入

【厚生年金勘定】
  ・保険給付費: 24.99兆円
  ・基礎年金勘定へ繰入: 20.33兆円 ← RS対象外
  ・実施機関交付金: 4.76兆円 ← RS対象外
  ・業務勘定へ繰入: 0.22兆円
```

厚生年金勘定の**20.33兆円**は、基礎年金の財源として基礎年金勘定へ移転されるため、RSシステムの「事業レビュー」対象外となる。

## 7. まとめ

1. **厚生年金勘定の差額25.40兆円の正体が完全に解明**
   - 80%（20.33兆円）は基礎年金勘定への財源移転
   - 19%（4.76兆円）は共済組合への交付金
   - これらは事業レビュー対象外の「会計間繰入・交付金」

2. **基礎年金勘定はほぼ完全に一致**
   - 差額わずか0.48兆円（1.7%）
   - 給付費そのものは完全一致

3. **RSシステムとMOFデータの役割が明確化**
   - RSシステム: 事業レビュー対象の「保険給付費」に特化
   - MOFデータ: 会計全体の資金の流れを網羅（給付 + 繰入 + 交付金）

4. **データ活用の方向性**
   - RSシステム: 個別事業の給付状況・支出先の把握
   - MOFデータ: 年金制度全体の財源構造・資金の流れの把握
   - 両者を組み合わせることで、年金制度の全体像を多角的に分析可能
