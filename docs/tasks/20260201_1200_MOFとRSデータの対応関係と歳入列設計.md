## 3. データの対応関係

### 3.1 一般会計の対応

```
【MOF一般会計歳出】114.4兆円
├─ 財務省: 35.5兆円（国債費）→ RSに含まれない
├─ 総務省: 16.9兆円（地方交付税）→ RSに含まれない
└─ その他: 62.0兆円 → RS一般会計71.5兆円の一部

【RS一般会計】71.5兆円
└─ 事業として計上された予算のみ
```

**差額の内訳**:
- MOF: 114.4兆円
- RS: 71.5兆円
- 差額: 42.9兆円 = 国債費(25.4兆円) + 地方交付税(16.9兆円) + その他調整

### 3.2 特別会計の対応

```
【MOF特別会計】444.6兆円（重複含む）
├─ 年金: 99.5兆円 ──────→ RS厚労省(特別会計分)の一部
├─ 労働保険: 9.0兆円 ───→ RS厚労省(特別会計分)の一部
├─ 国債整理基金: 239.5兆円 → RSに含まれない（借換債）
├─ 交付税配付金: 51.2兆円 → RSに含まれない
└─ その他: 45.4兆円

【RS特別会計】77.1兆円
└─ 事業として計上された予算のみ
```

### 3.3 府省庁レベルの対応

## 4. サンキー図設計選択肢

### 選択肢A: RSベース（推定財源）

```
【歳入（推定）】         【予算総計】      【府省庁】→【事業】→【支出先】

租税      約45兆円  ───→                 ├─ 厚労省 92.8兆円 → ...
公債金    約24兆円  ───→  151.1兆円  ───→├─ 経産省 12.3兆円 → ...
保険料    約37兆円  ───→                 ├─ 国交省 11.1兆円 → ...
その他    約45兆円  ───→                 └─ その他 34.9兆円 → ...
```

**按分ロジック**:
- RS一般会計(71.5兆円)にMOF一般会計の財源比率を適用
- RS特別会計(77.1兆円)にMOF特別会計の財源比率を適用

**メリット**: 事業・支出先まで一貫性がある
**デメリット**: 財源額は推定値

### 選択肢B: MOFベース（正確な財源、事業接続なし）

```
【歳入（MOF）】          【歳出（MOF所管別）】

租税      68.5兆円  ───→├─ 財務省     35.5兆円（国債費）
公債金    35.6兆円  ───→├─ 厚労省     33.2兆円
その他    10.3兆円  ───→├─ 総務省     16.9兆円（交付税）
─────────────────────   └─ その他     28.9兆円
合計     114.4兆円       合計        114.4兆円

※ 事業・支出先への接続なし（別画面で表示）
```

**メリット**: 財源データが正確
**デメリット**: 事業単位のデータと接続できない

### 選択肢C: ハイブリッド（両方表示）

```
【財源（MOF）】    【一般会計】   【RS予算】    【事業】→【支出先】
                   ↓              ↓
租税    68.5兆円 → 一般会計 ────→ RS一般会計  → ...
公債金  35.6兆円 → 114.4兆円      71.5兆円
その他  10.3兆円 →
                   差額42.9兆円 = 国債費+地方交付税（事業外）

【財源（MOF）】    【特別会計】   【RS予算】    【事業】→【支出先】
                   ↓              ↓
保険料  50.2兆円 → 年金等 ─────→ RS特別会計  → ...
繰入    32.5兆円 → 99.5兆円       77.1兆円
その他  ─────────→
```

**メリット**: 両方のデータを可視化、差異を明示
**デメリット**: 表示が複雑
