## 発見事項（今後の改善候補）

1. **OpenRouter の一過性障害でリトライなし**: `choices[0].message` 欠落（HTTP 200）で即 502。1回だけの自動リトライを入れる価値あり（v1 実装既存の挙動で WP1 のスコープ外。次の改修時に）
2. **`get_quality_scores` への pid 引き渡し経路**: 13件マッチ中10件しかスコア比較されなかった。run_sankey_query の summary が上位10件しか返さないため。「全件比較して」への対応は summary の pid リスト拡張か、フィルタ条件を直接受ける quality ツールへの改修が候補
3. レイテンシはフィルタ構築系で 15〜27秒。WP2-2（ストリーミング/進行イベント）の動機を定量的に裏付けた
