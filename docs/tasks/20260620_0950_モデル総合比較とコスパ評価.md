**日時**: 2026-06-20 09:50 (JST)
**対象**: 2024年度 AIスコア確定生成に使うモデルの選定。品質・レート・コストを横断比較し、本番方針とコスパ系候補を整理する。

## 3. コスパ評価（品質×価格）

| 観点 | 推奨 | 理由 |
|------|------|------|
| **2025と基準完全一致（本番）** | **gemini-3.5-flash** | 同一モデルでしか年度横断の整合は取れない。ただし$63〜$181 |
| **コスパ最優先** | **deepseek-v3.2 / v4-flash** | $2〜$5 で MAE 1.8・**バイアスほぼゼロ**。3.5-flashの約1/15〜1/30 |
| **無料で傾向把握** | gemini-2.5-flash(20RPD少量) / 3.1-flash-lite(500RPD全件・甘め別分布) | $0。確定不可・参考値 |

### DeepSeek を選ぶ場合の注意

- DeepSeek は 3.5-flash と **MAE 1.78** ＝ 2024 を deepseek、2025 を 3.5-flash のまま混在させると**年度間で基準がズレる**（他の代替モデルと同じ制約）。
- ただし DeepSeek は安いので、**2024・2025の両方を deepseek-v3.2 で再生成すれば ~$9 で年度横断の基準を揃えられる**（3.5-flash基準を捨てて、より安い一貫基準に置き換える案）。バイアスが小さい分、絶対値の解釈もしやすい。
- 反面、DeepSeek は分散が大きめ（同一事業でのブレ）。確定採用前に標本を増やした再現性確認が望ましい。

## 4. 方向性（現時点の結論）

- **本番（2025整合を保つ）**: `gemini-3.5-flash`。コストは $63〜$181 を見込む（thinking上振れ込み）。課金 Tier1（上限$250）で当日完了。
- **コスパ系の有力候補**: `deepseek-v3.2`（~$4.6・低バイアス・JSON安定）。「2024のみ」なら基準ズレ、"両年 deepseek 再生成"なら ~$9 で一貫基準。要・標本拡大の再現性確認。
- **無料**: 品質一致（2.5-flash=20RPD）と全件可能（3.1-lite=500RPD・甘め）が両立しない。確定には使わない。

> 測定はいずれも軸E・n=18 の傾向値。確定モデル決定前に、候補モデルで `--limit 100` 程度の実走（実トークン・実分布）で本書の数値を補正することを推奨。
