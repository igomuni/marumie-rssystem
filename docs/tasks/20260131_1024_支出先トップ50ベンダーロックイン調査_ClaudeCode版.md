## 4. パターン別分類と考察

### 4.1 事務局/BPRモデル（22件）

このモデルでは、広告代理店・公益法人・独立行政法人が「事務局」として巨額の予算を受託し、実際の業務を再委託する構造が一般的です。

**特徴**:
- 占有率が高い（平均90%超）
- 再委託構造が複雑（2-4系）
- 最終受益者が不透明

**主要事例**:

### 6.1 調査結果のまとめ

本調査により、支出先トップ50における以下の実態が明らかになりました。

2. **事務局モデルの蔓延**: 広告代理店・公益法人が巨額の補助金事業を受託し、実務を再委託する構造が50社中20社以上で確認
3. **再委託の多層化**: 博報堂（4系）、キャッシュレス推進協議会（3系）など、3-4層の再委託構造が存在

### 6.3 提言

#### 提言1: 事務局モデルの透明化義務付け

博報堂やキャッシュレス推進協議会のような「事務局」型受託については、以下を義務付けるべきです。
- 再委託先の全法人名の公開（匿名禁止）
- 各層での手数料率の開示
- 再委託は原則2層まで（再々委託の禁止）

#### 提言2: 入札の競争性強化

入札者数2社以下、落札率95%超の案件については、以下の対策を講じるべきです。
- 予定価格の非公開または事後公開
- 総合評価方式の透明化（評価基準の事前公開）
- 1者応札理由の詳細開示（単なる「互換性」では不可）

#### 提言3: ITシステムのモジュール分割とクラウド化

富士通のハローワークシステム（74.1%占有）のような大規模ITシステムについては、以下の対策が有効です。
- 機能別モジュール分割による複数社発注
- ガバメントクラウド（Gov-Cloud）への段階的移行
- オープンスタンダードの採用によるベンダーロックイン回避

#### 提言4: 5-2 CSV開示基準の強化

現行の5-2 CSV（支出ブロックのつながり）では、再委託先が「民間企業」「株式会社Aほか」等で匿名化されています。透明性向上のため、以下の改善が必要です。
- 全再委託先の法人名・法人番号の開示
- 再委託金額の開示（現状は金額記載なし）
- 再委託理由の開示
