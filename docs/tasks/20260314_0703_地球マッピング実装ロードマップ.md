## Phase 2: Globe API + 府省庁マーカー配置 ✅

**PR**: #90

**ゴール**: APIから省庁データを取得し、球面上にFibonacci格子で点を配置して表示する

**成果物**:
- `app/api/map/globe/route.ts` — 府省庁レベルの球面データ配信
- GlobeView に府省庁マーカー（ラベル付きの点）を描画

**内容**:
- API: `rs2024-structured.json` から府省庁ごとの `totalSpending`, `areaFraction`, `projectCount` を計算
- Fibonacci格子で省庁の `seed [lon, lat]` を算出
- 省庁ごとに色相を割り当て（HSLのH値を等分割）
- クリック・ホバーで省庁名・金額を表示

## Phase 3: Icosphere面積比例マッピング ✅

**PR**: #91

**ゴール**: 府省庁が実質支出額に比例した面積で球面を分割する

**成果物**:
- GlobeView をicosphere方式に全面改修
- API に小規模省庁の集約ロジックを追加
- [府省庁別立体角一覧](20260314_1310_府省庁別立体角一覧.md)

**内容**:
- icosphere（正二十面体の再帰4分割、レベル6 = 81,920面）を生成
- 面を北極→南極の緯度帯順にソートし、支出額比率で連続割り当て
- レベル6で0面になる下位5省庁を「その他」に集約（33省庁 + その他 = 34エントリ）
- 単一メッシュ＋頂点カラーによる高速描画（`toNonIndexed()` で面ごとの色分け）
- ホバーハイライト（全面色変更）、ツールチップ（面数表示付き）
- ラベルを各省庁の北端境界に配置、ズーム連動スケーリング
- Zoom率UI（デフォルト1.00x、nearクリップ防止 `minDistance = SPHERE_RADIUS + camera.near`）

**経緯**:
- 球面Voronoi → multi-seed Voronoi → icosphere緯度帯割り当てと方式を試行
- Voronoiは等面積に収束する性質があり、厚労省65.7%の極端な偏りと相性が悪かった
- 緯度帯割り当てが面積比例を正確に実現でき、描画性能も良好

## Phase 4: 領域割り当てアルゴリズム改善 ✅

**PR**: #92

**検討資料**:
- [Globe領域割り当てアルゴリズム検討](20260314_1553_Globe領域割り当てアルゴリズム検討.md)
- [Globe浮島問題の原因調査・サッカーボール展開方式設計](20260315_0254_Globe浮島問題の原因調査.md)
- [Globe Phase 4B 再帰二分割ツリーマップ実装ドキュメント](20260315_0857_Globe再帰二分割ツリーマップ実装ドキュメント.md)
- [Globe Phase 4C 多面体シード配置実装ドキュメント](20260315_1554_Globe多面体シード配置実装ドキュメント.md)

**ゴール**: 全33府省庁を連続領域（1コンポーネント）で球面に配置し、境界を滑らかにする

**成果物**:
- GlobeView: 多面体シード配置方式（4段階バッチ処理）に全面改修
- Globe API: 小規模省庁集約を廃止、全33府省庁をicosphere level 7（327,680面）で配置
- 検証スクリプト `scripts/globe-assignment-check.mjs`

**進化の経緯**:
1. **Phase 4A: パンゲア配置**（30bf674） — 海（厚労省）を南半球に集め、残りを北半球パンゲアに。浮島問題が発覚
2. **Phase 4B: 再帰二分割ツリーマップ**（d67962a） — 2D lon/lat座標で矩形分割。ストライプ問題あり
3. **Phase 4C: 多面体シード配置**（f97771e） — 3D座標での立方体→正八面体→立方八面体→海岸線BFS。全33府省庁1コンポーネント達成

**多面体シード階層**:
| バッチ | 多面体 | ゾーン数 | 方式 |
|--------|--------|---------|------|
| 1 | 立方体 | 8 | 3D再帰二分割 |
| 2 | 正八面体 | 6 | 軸交点Voronoi |
| 3 | 立方八面体 | 12 | 辺中点Voronoi |
| 4 | — | 6 | 海岸線BFS |

**試行した方式（不採用）**:
- 3軸（三角形）/ 5軸（五角形）/ 6軸（六角形）分割 → 断片化が許容範囲外
- 正六角形パッキング（Voronoi / 容量制約 / 貪欲配置） → 球面投影との根本的不適合
- 正六角形5大陸ループ → 大きな隙間が残る
- 45°回転軸分割 → ゾーン中心が交点に収束しない

**ハーネス確認項目**:
- [x] 全33府省庁が1コンポーネント（断片化ゼロ）
- [x] 海（厚生労働省）も1コンポーネント
- [x] 面積比例の精度維持（±2%以内）
- [x] 境界滑らか化（5イテレーション局所スワップ）
- [x] フレームレートが低下していない
