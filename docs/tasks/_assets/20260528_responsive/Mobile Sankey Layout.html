01 — Top bar (§3)

検索 + 年度ピル + 設定の3要素のみ

年度は閲覧コンテキストの起点なので常時可視（ピル44h、タップで年セレクター開閉）。 TopN/フォント/表示モード等は ⚙ から1シートに集約。 390px幅で 12 + flex搜索 + 8 + 60ピル + 8 + 44⚙ + 12 = フィット。

02 — Side panel (§4)

bottom-sheet と fullscreen の2案を並置

390px (iPhone等) は bottom-sheet。Sankey連続性を維持しつつ詳細閲覧可。 360px (Android狭幅) は本体閲覧が苦しくなるため fullscreen overlay に切替。 どちらも selectedNodeId / コンテンツは共通、プレゼンテーション層のみ差替え。

03 — DisplayMode (§1)

自動判定 + 手動上書き

既定は幅 + pointer:coarse で自動判定。 設定シート末尾に「自動 / コンパクト / 標準 / プレゼン」を露出し、URLクエリ・localStorage に保存。 プリセット拡充は後続タスクに譲るが、基盤UIは本モックで明示。
