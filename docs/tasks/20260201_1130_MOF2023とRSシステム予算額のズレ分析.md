## 6. ズレの原因分析

### 6.1 一般会計のズレ（MOF 114.4兆円 vs RS 71.5兆円）

**差額: 42.9兆円**

| 原因 | 金額 | 説明 |
|------|------|------|
| **国債費** | **約25.4兆円** | 財務省予算の大部分。国債の利払い・償還は「事業」ではないためRSに含まれない |
| **地方交付税** | **約16.9兆円** | 総務省予算の大部分。地方への移転であり中央政府の「事業」ではない |
| その他調整 | 約0.6兆円 | 端数調整等 |
| **合計** | **約42.9兆円** | |

**検証**:
- MOF一般会計: 114.4兆円
- 国債費: -25.4兆円
- 地方交付税: -16.9兆円
- 調整後: **72.1兆円** ≒ RS一般会計 **71.5兆円** ✅

### 6.2 特別会計のズレ（MOF 444.6兆円 vs RS 77.1兆円）

**差額: 367.5兆円**

| 原因 | 金額 | 説明 |
|------|------|------|
| **国債整理基金** | **239.5兆円** | 借換債の発行・償還。会計上の処理で実質的な「支出」ではない |
| **会計間繰入** | **約80兆円** | 特別会計同士の資金移動（重複計上） |
| **財政投融資** | **約25兆円** | 融資のため「支出」として計上されない |
| その他調整 | 約23兆円 | |

**RSの特別会計77.1兆円の内訳**:
- 年金特別会計の給付費等: 約50兆円
- その他特別会計の事業費: 約27兆円

## 7. 歳入列追加への影響

### 7.1 推奨アプローチ

RSシステムの予算総額151.1兆円に対応する財源として表示するには:

```
歳入（財源）151.1兆円
├─ 租税（税金）: 約68.5兆円（45%）
├─ 公債金（国債）: 約35.6兆円（24%）
├─ 社会保険料: 約37兆円（24%）※RSに含まれる分のみ
└─ その他: 約10兆円（7%）
```

### 7.2 注意点

1. **MOF一般会計の財源比率をRS一般会計に適用**
   - 租税60% + 公債金31% + その他9%

2. **RS特別会計の財源は別途計算**
   - 保険料70% + 一般会計繰入30%（年金特別会計の場合）

3. **重複計上の回避**
   - 一般会計→特別会計繰入は二重計上しない
